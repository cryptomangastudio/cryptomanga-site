# 小説プラットフォーム / 縦型原作事業(検証中)

「AIオッケーなカクヨム型小説投稿プラットフォーム(マンガ・ショートアニメ・ドラマ原作の供給源)」構想の検証結果と、そこから組み替えた事業の土台。

> このフォルダも kindle/ と同じ流儀: 「勝てます」と煽らず、**徹底リサーチの検証結果に基づき、できること・できないことを正直に**置く。

## 先に結論(徹底リサーチの回答)

- **「AI許容の汎用投稿プラットフォーム」= 勝ち筋なし。**
  1. 「AIオッケー」はもう差別化にならない(カクヨム・なろうとも2026年に「開示義務つき容認」へ移行済み。海外もRoyal Road・KDPが許容済み)
  2. AI許容を看板にすると品質シグナルを毀損する(中国・番茄小説は読者反発でAI投稿を月10万件却下する側に転じた)
  3. 後発の汎用投稿サイトは大資本でも全滅してきた(LINEノベル1年で撤退、MAGNET終了、ネオページ苦戦)。カクヨムはKADOKAWAの出口があって成立している
- **「縦型ショートドラマ/ショートアニメ向けの原作供給」に組み替えれば条件付きで勝ち筋あり。**
  - 需要は実在: 中国の短劇市場約1兆円、日本は世界2位の高課金市場なのに原作パイプラインが細く、日販・トーハン・BUMP・Nola・DMMが原作公募・スカウトに奔走中
  - 勝ちパターンは「場」ではなく「原作の目利き・編集・出口を握る垂直統合」(アルファポリス: 売上136億円の約8割がコミカライズ)
  - 文化庁見解により完全AI生成には著作権が立たない=買い手がつかない。「AI補助+人間編集+証跡」なら権利化でき、これはkindle事業で確立済みの運用 → **AI量産勢が入れない堀になる**
- **推奨: プラットフォームを作らない参入(段階設計)**
  - 第0段階(投資ほぼゼロ・3〜6カ月): 自社制作の縦型原作をBUMP公募・Nolaシナリオ甲子園・DMM持ち込み等に出して需要と単価を実測
  - 第1段階(数万円): 当たった型を複製し「原作カタログ(レーベル)」を公開、制作会社と直接取引
  - 第2段階: 引き合いが実在したときだけ、投稿受付(ミニプラットフォーム)や共催コンテストと資金調達を検討
  - 停止ライン: 6カ月で採用0件・引き合い0件なら第1段階に進まない
