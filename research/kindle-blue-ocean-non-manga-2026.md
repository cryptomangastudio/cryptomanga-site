# 日本Kindle(KDP)ブルーオーシャン調査 第2弾: マンガ以外の抜け穴 2025–2026

**作成日**: 2026-07-11
**位置づけ**: `kindle-blue-ocean-2026.md`(第1弾・マンガ中心)の続編。今回は**マンガ以外の形式**(テキスト実用書、読み物、ペーパーバック/ローコンテンツ本、海外市場)を対象に、4系統の並列調査(約120情報源)を統合した。

---

## 1. エグゼクティブサマリー

マンガ以外で「需要が実証されているのに個人出版の供給がほぼゼロ」の抜け穴は、大きく**5つの構造的な歪み**から生まれている:

1. **制度変更の直後・直前**(2026年10月「106万円の壁」撤廃、2025年登録販売者の難化、2022年開始のドローン国家資格)— 需要が新しすぎて書籍供給が追いついていない
2. **KDP自体のルール変更直後**(2026年4月に日本で「標準カラー印刷」解禁 = 鉛筆で書き込める紙+3.5円/ページ)— 塗り絵・カラードリルの採算が数ヶ月前に激変したばかりで、気づいている参入者が少ない
3. **シニア市場の需給ギャップ** — 需要側は拡大(文字拡大ニーズ55.4%、50-60代の過半数がKindle利用、脳トレは介護施設のリピート消費)なのに、個人出版者はSNS世代向けばかり作っており供給がほぼゼロ
4. **「当事者向け」の空白** — 業界解説書はあるのに当事者(トラックドライバー、コールセンター、保育士)向けのお金・転職本がない
5. **日本語ネイティブ品質の空白** — 日本語ローコンテンツ本の現供給は海外セラーの機械翻訳品が中心。逆に英語圏では「本物の日本人が作った日本もの」が空白

## 2. 優先順位付き「抜け穴」リスト(マンガ以外)

評価軸: 需要の実証度 × 供給の薄さ × タイミングの窓 × スタジオ適合(デザイン・編集力)。

### ★S-1: シニア向け脳トレ・ペーパーバック(昭和レトロ間違い探し/大活字ナンプレ/漢字・計算ドリル)

- **需要の実証**: 川島隆太監修「毎日脳活」等の商業ムックが量販、「義母に20冊以上購入」等のリピート消費、介護施設のレク需要、楽天「高齢者 間違い探し」322件ヒット。75歳免許更新の認知機能検査対策ドリルという明確な検索需要も存在。
- **供給の薄さ**: 商業ムックは厚いが、**日本語KDP個人出版の脳トレ・パズル本はごく少数。実践者の収益報告が「存在しないこと」自体が未開拓のシグナル**。
- **採算**: 白黒は印刷原価が最安(固定206円+2円/ページ)。1,100〜1,500円設定でロイヤリティ60%を確保(2025年6月改定で1,000円未満は50%に低下したため1,000円以上が必須)。
- **タイミングの窓**: **2026年4月に日本で「標準カラー印刷」解禁**(206円+3.5円/ページ、白黒と同じ筆記可能な非コート紙)。カラー見本付き塗り絵・カラー脳トレなど、これまで採算が合わなかった商品が数ヶ月前から可能になったばかり。
- **差別化**: 大活字・太枠・めくりやすさなど「高齢者仕様」の設計+イラスト内製力。昭和レトロ画風はスタジオの作画力がそのまま活きる(マンガを描かずに絵の力を使える)。

### ★S-2: 制度改正の先回り実用書 — 「106万円の壁」撤廃(2026年10月施行)

- **需要の実証**: パート・短時間労働者**数百万人**が直接影響する年金制度改正。企業規模要件(51人以上)の段階的撤廃も同時進行。
- **供給の薄さ**: 施行前のため一般向け書籍が**ほぼゼロ**。
- **タイミング**: 施行3ヶ月前の今が出版の最適タイミング。FP・社労士資格がなくても公的資料ベースで執筆可能。
- **同系統の第2弾**: 在職老齢年金の改正(2026年4月施行、月62万円まで減額なし、新たに20万人が全額受給)、女性活躍推進法101人以上義務化(2026年4月)。**「制度改正カレンダー→施行6ヶ月前に出版」を仕組み化すれば反復可能なパイプラインになる**。

### ★S-3: 難化・新設資格の勉強法(体験記型)

| 候補 | 需要規模 | 空白の中身 |
|---|---|---|
| **登録販売者・難化対応** | 受験者約5.9万人/年。2025年度合格率46.7%→40.7%に急落、首都圏38.7% | 既存テキストは難化前提で書かれていない。「難化後の時間配分」「合格率の高い県への越境受験(都道府県で31ポイント差)」を扱う本が皆無 |
| **ドローン国家資格(二等)独学** | 取得者3万人超、月2,000人ペース増の新資格(2022年開始) | スクール(数十万円)誘導記事ばかりで中立な独学対策書がほぼゼロ。「スクール代を浮かせたい」という金銭動機が明確 |
| **通関士・独学記録** | 受験者6,322人/年、合格率15.1%。2024年合格者半減が業界ニュース化 | 独学体験記はnoteレベルのみで書籍供給が非常に薄い |
| 気象予報士・文系独学 | 年2回×約4,000人、合格率4〜5% | 挫折率が高く「心の支え」需要。書籍化は僅少 |
| 賃貸不動産経営管理士 | 受験者31,792人/年 | 「宅建との掛け合わせ勉強法」が薄い |

- **成功パターンの型**(実例分析より): (1)タイトルに書き手の属性ハンディを明示(50代/文系/主婦)、(2)数値を入れる(3ヶ月/一発)、(3)失敗過程を隠さない記録型、(4)60〜100頁で十分。

### ★A-1: シニア向け短編読み物・大活字設計(KUテキストの空白地帯)

- **需要**: 電子書籍のメリット1位が「文字拡大」55.4%。50〜60代電子書籍ユーザーの過半数がKindle利用。終活・自分史は幻冬舎ルネッサンスが講座化するほどの需要。
- **供給**: **「時代小説×シニア×個人KDP」「昭和回顧エッセイ」の実践情報・競合報告がほぼ存在しない=空白地帯**(収益公開が見つからないこと自体が先行者不在の傍証)。
- **KU相性**: 短編連作+合本のKENP最大化定石がそのまま使える。リスクはシニアのKU加入率が未知数な点のみ。

### ★A-2: 「当事者向け」職業×お金の実用書

- **トラックドライバー**: 就業者約80万人の大職種なのに当事者向け書籍が空白。2024年問題で労働時間+17%・年収-4〜12%(公的データあり)という切実な検索需要が発生中。
- **コールセンター×メンタル**: 大量雇用・高離職率なのに書籍がほぼ存在しない。
- **保育士の辞め方・介護士の処遇改善加算**: 転職メディアのWeb記事は大量(=検索需要の実証)だが書籍は空白。

### ★A-3: 海外市場 — 「本物の日本人が作った日本もの」(英語圏)

- **オノマトペ×イラストワークブック**: 「教科書でほぼ扱われないが日本語の核心」と英語圏学習者に認知済み。イラスト付き実用ワークブック形式はほぼ空白。**イラスト制作力が直接活きる最有力候補**。
- **漢字・和柄のBold and Easy塗り絵**: 太線・シニア/初心者向け塗り絵は2025-26英語圏KDPの#1成長トレンド($8.99〜12.99)。ニッチ特化型は競合1,000未満で月$500〜3,000の報告。**翻訳負担ほぼゼロ**。
- **ビジネス日本語/敬語ワークブック**: GaijinPotが「上級・ビジネス向け教材の選択肢が少ない」と指摘。ネイティブの敬語知識が最大の武器。
- **和風デザインのローコンテンツ本**(俳句ジャーナル、御朱印帳風トラベルログ、一日一禅ジャーナル): 汎用プランナーは飽和だが「和風美意識×特定用途」は薄い。
- **逆説的な追い風**: 英語圏の「日本もの」上位本(ikigai/wabi-sabi等)は著者のほとんどが**非日本人またはAI量産ペンネーム**。「日本人ネイティブが作った」ことをブランド化できる。
- **実務**: 日本アカウントのまま全世界配信可(移行はレビュー消失リスク)。W-8BENは「TIN例外国」チェックで源泉税0%(マイナンバーは使用不可)。米国は2026年7月7日から70%枠が$2.99〜12.99に拡大。KDPセレクト必須なのは日本・伯・墨・印のみで、**米英では独占なしで70%が取れる**。

### ★A-4: 非18禁のソフトBL/TL短編シリーズ

- **根拠**: BLは**「素人が商業に勝てる」と複数報告される稀有なフィクションジャンル**(個人作品がランキング上位、1タイトル月8万円の報告、Kindle総合61位到達例)。
- **条件**: 18禁を避けて通常検索露出を維持することが絶対条件(アダルト判定=検索非表示・海外IP遮断で商品として死ぬ)。3〜5冊+合本のシリーズ設計。表紙品質への投資が必須(スタジオのデザイン力が活きる)。
- **注意**: 収益分散が大きい(年数千円の報告もある)ため、多点テストの1路線として扱う。

### B枠(小粒・実験向き)

- **趣味特化ログブック(釣行記録・登山記録・推し活記録・栽培記録)**: 現供給は海外セラーの機械翻訳品が中心で品質ギャップが大きい。釣果記録アプリが製本サービスを出すほど紙需要は実在。1点あたりは小さく多品種展開が前提。
- **職業特化の業務手帳**(サロン顧客カルテ、送迎記録など): トリマー予約帳の海外製が既に売られている=需要実在。
- **熟年離婚チェックリスト**(第1弾から継続): 弁護士SEO記事大量=需要実証、チェックリスト型書籍は空白。
- **高価格実用書+特典テンプレPDF型**(電子): テンプレ本単体は成立しないが、1,500円級の解説書に「読者特典PDF」を付ける形が定石。

## 3. 避けるべき領域(マンガ以外の追加分)

| 領域 | 理由 |
|---|---|
| 汎用ノート・ジャーナル・日記帳(日本語) | 海外セラーが機械翻訳で大量投入済み+100均・商業文具と価格競争で必敗 |
| 家計簿 | 永岡書店等の商業品が800円台で支配。改定後KDPは1,000円未満で利益が出ない |
| 血圧手帳・お薬手帳 | 薬局・自治体の無料配布文化で有料需要が構造的に薄い |
| 御朱印帳 | 蛇腹製本が本質。KDPの無線綴じでは商品として成立しない |
| 子ども向け低価格ドリル | 無料プリント文化+うんこドリル級の商業品。500〜700円帯はKDPの採算外 |
| 英語圏向け汎用ローコンテンツ(数独・ジャーナル) | 英語圏では飽和が公知。後発参入は非推奨 |
| 電子書籍の塗り絵・書き込みワークブック単体 | 媒体特性上不成立(塗り絵は紙一択) |
| 18禁官能小説(テキスト) | アダルト判定=検索非表示・年齢ゲート・海外遮断。「1000ページも読まれない」実名失敗報告あり。ハイリスク中リターン |
| 電子絵本 | ページ数が少なくKENP構造と相性最悪(1冊読了でも数十円)。AI作画参入で供給過剰化の兆し |
| ikigai/wabi-sabiテキスト本・汎用ひらがな練習帳・日本旅行ガイド(英語圏) | いずれも飽和+AI量産乱立 |
| なろう系KDP移植 | 定石化済みで競合過多。KDPセレクト中のWeb全文公開は独占違反になる点も要注意 |
| オーディオブック(当面) | ACX日本未対応、audiobook.jpは実質法人向け。ヒット作ができた後の二次展開枠として保留 |

## 4. 収益構造の要点(マンガ以外で変わる点)

- **ペーパーバックの改定(2025年6月)**: ¥1,000未満はロイヤリティ60%→50%。**紙で出すなら1,100円以上の値付けが実質必須**。白黒印刷コストは固定206円+2円/ページ(100ページ≒406円、1,100円設定で1冊あたり約250円の利益)。
- **標準カラー解禁(2026年4月・日本)**: 206円+3.5円/ページ、筆記可能な非コート紙。72ページフルカラーで原価458円。塗り絵・カラードリルの参入窓が開いたばかり。
- **テキスト電子書籍の最適ボリューム**: 2〜4万字(KENPC150〜300ページ)が読了率と収益のバランス点。60〜100頁の薄い本で十分売れる(成功実例多数)。
- **分冊+合本の定石**: 短編シリーズ(統一シリーズ名+巻番号)→3〜5冊で「合本版」。ただし水増し分割・過度なコンテンツ再利用は公式ガイドライン違反(削除・アカウント停止リスク)。
- **ペーパーバックの意外な比重**: 月10万円超のクライアントで「印税の4割以上がペーパーバック」という支援業者の報告あり(未検証)。電子と紙の同時刊行を標準にすべき。

## 5. 実行順序の提案(第1弾のプレイブックと統合)

1. **今すぐ(〜2026年9月)**: 「106万円の壁」本を最優先で制作(施行前の3ヶ月が勝負)。並行してシニア脳トレ・ペーパーバック第1弾(白黒間違い探し or 大活字ナンプレ)を1冊テスト刊行。
2. **第2波(〜2026年内)**: 資格系(登録販売者難化対応、ドローン独学)とシニア短編読み物をテスト投入。反応の良い路線を特定。
3. **第3波(2027年〜)**: 当たった路線のシリーズ化+合本。英語圏(オノマトペWB、漢字塗り絵)は翻訳負担が小さいものから並行実験。制度改正カレンダーを常設パイプライン化。
4. **共通ルール**: 全点「電子+ペーパーバック同時刊行」「KDPセレクト登録(日本)」「参入前にAmazon手動確認: 競合100冊未満・上位レビュー10〜30件以下」。

## 6. 残る不確実性

- 日本語ローコンテンツ/シニア脳トレの「実践者収益報告の不在」は、(a)未開拓、(b)実は売れない、の両解釈が可能。**小ロットのテスト刊行で実売検証するまで断定不可**(ただし商業ムックのリピート消費と介護施設需要という需要側の実証は強い)。
- シニア層のKU加入率・Kindle購買行動の一次データは未取得。
- 資格系の受験者数は一次データ(公式統計)で確認済みだが、「受験者がKindleで勉強法本を買うか」の転換率は未検証。
- Amazonページの直接取得が遮断されているため、供給冊数・レビュー数は間接確認。**企画確定前に必ず手動でAmazon検索して「競合100冊未満・上位レビュー10〜30件以下」を確認すること**。
