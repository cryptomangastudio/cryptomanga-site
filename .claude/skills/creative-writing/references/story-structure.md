# プロット・構成の作り方

## 0. まずログラインを1〜2文で作る

「〈欠落を抱えた主人公〉が〈目的〉を求めて〈障害〉に立ち向かう話」の形で言い切る。
ログラインが作れない=まだ物語の芯がない、という診断に使う。書きながら詰まったら、ここに戻る。

- 悪い例:「異世界に転生した高校生が冒険する話」(誰の何が問題なのか不在)
- 良い例:「他人の期待に応え続けて自分の意志を持てなかった高校生が、異世界で"失敗しても死なない"ことを知り、初めて自分の意志で選択する話」

ログラインには**皮肉(アイロニー)**が入っていると強い。「消防士なのに火が怖い」「人の心が読めるのに、いちばん大事な人の心だけ読めない」のように、主人公の属性と課題が衝突している形。

## 1. 起承転結(短編・単話向き、日本の物語文法として自然)

- **起**:日常・世界観・主人公の欠落を提示する。掴みが命。マンガなら1〜3ページ、小説なら数段落で「これは何の話か」を読者に予感させる。
- **承**:欲求が明確化し、行動を始める。小さな障害・伏線を積む。
- **転**:想定外の事態。価値観が揺さぶられる。ここが弱いと物語全体が平坦になる。「転」は単なる事件ではなく、主人公の**認識や関係性が反転する**瞬間であるべき。
- **結**:転を受けての着地。欠落がどう変化した(埋まった/形を変えた/あえて埋まらなかった)かを見せる。

起承転結は「4つの均等なパート」ではない。実務上は「起」を薄く速く、「転」に厚みを持たせるのが現代の商業マンガ・WEBTOONの主流(掴みが早いほど離脱されにくい)。

### 32ページ読切での配分の目安

- 起:P1〜5(主人公と欠落の提示、事件の発生)
- 承:P6〜16(行動と障害、関係の深まり)
- 転:P17〜26(最大の危機と認識の反転。クライマックスの見開きはこの帯に置く)
- 結:P27〜32(決着と変化の提示。ラスト1〜2ページは余韻に使う)

## 2. 三幕構成(長編・シリーズ全体の設計向き)

1. **第一幕(設定 ~25%)**:日常世界、主人公の欠落、きっかけ(インサイティング・インシデント)、決意のポイント(Point of No Return)。
2. **第二幕(対立 ~50%)**:目的への行動と障害の連続。中盤(ミッドポイント)で「勝ったと思ったら実は状況が悪化した/価値観が変わる」転換を置く。第二幕後半で「オールイズロスト(万事休す)」— 主人公が最も無力になる場面を作る。
3. **第三幕(解決 ~25%)**:オールイズロストを乗り越える決断、クライマックス、そして欠落がどう変わったかを示す新しい日常。

**ミッドポイントとオールイズロストを先に決める**と、中だるみしやすい第二幕が組み立てやすい。ミッドポイントの型は主に2つ:

- **偽りの勝利**:目的を達成したかに見えるが、実はより大きな問題の入口だった(例:敵幹部を倒したら、その敵こそが街を守っていた)
- **偽りの敗北**:すべて失ったかに見えるが、そこで初めて本当のNeed(欠落)に気づく

## 3. 感情曲線で考える(構造論の補助線)

構成テンプレートに沿っていても面白くならないときは、**読者の感情の上下**をグラフとして描いてみる。

- 「上がる(希望・爽快)」と「下がる(不安・絶望)」が交互に来ているか。上がりっぱなし/下がりっぱなしの区間が長いと単調になる。
- 落差が大きいほど感動も大きい。クライマックスの直前に最も深い谷(オールイズロスト)を置くのはこのため。
- 各話単位でも同じ。1話の中に最低1回は感情の反転を入れる。

## 4. ヒーローズジャーニー(神話的・冒険譚向き。使いどころを選ぶ)

日常→冒険への誘い→拒絶/躊躇→師との出会い→境界を超える→試練の連続→最も深い洞窟(最大の試練)→復活→宝を持って帰還。
王道異世界・冒険ものと相性が良いが、そのまま使うとテンプレ感が強く出るので `genre-playbook.md` の「外し方」と組み合わせる。

## 5. 連載マンガ/WEBTOONの話数設計

単発の三幕構成とは別に、**連載として続く構造**を考える必要がある。

- **各話の終わり=引き(フック)**。次話が読みたくなる問い・危機・驚きで終える。「1つの疑問に答えつつ、新しい疑問を残す」が理想。
- **引きの型を使い分ける**(毎回同じ型だと読者が慣れる):
  1. **危機型**:キャラクターが物理的・社会的ピンチに陥ったところで切る
  2. **謎提示型**:新しい謎・違和感を見せて切る(「その写真に写っていたのは——」)
  3. **決意型**:キャラクターが重大な決断を宣言して切る(実行は次話)
  4. **反転型**:見えていた状況が覆る一撃で切る(裏切り、正体バレ)
  5. **感情型**:大きな感情の高まり(告白、涙、再会)の瞬間で切る
- **章(アーク)単位で三幕構成を入れ子にする**。1話単位の小さな起伏の上に、5〜15話程度の中アーク、さらに作品全体の大きな弧を重ねる。
- **第1話は「作品全体の縮図」にする**。作品の魅力(バトルならバトルの快感、恋愛なら関係性の甘さ/もどかしさ)を必ず1話の中で1回は本気で見せる。「本番は3話から」は現代の連載では通用しない。
- **伏線は「気づかれない程度に早く、忘れられない程度に近く」**置く。目安:回収の3〜7話前が読者の記憶に残りやすい。長期伏線は中間で1回「思い出させる」描写を挟む。

### 話数プロット(箱書き)のテンプレート

```
■第◯話「タイトル」
- 目的:この話で読者に届ける感情(例:主人公とヒロインの距離が縮まる高揚)
- あらすじ:3〜5行
- 開幕:前話の引きの回収をどう処理するか
- 山場:この話のいちばんの見せ場(絵になる瞬間を1つ言語化する)
- 引き:型(危機/謎/決意/反転/感情)と内容
- 仕込み:この話で張る伏線・積む関係性
```

## 6. 中だるみ・薄いプロットの診断チェック

- 主人公が受け身のまま話が進んでいないか(事件に反応するだけで、能動的な選択をしていないか)
- 「転」で価値観や関係性が本当に変化しているか、単なるイベント追加になっていないか
- 障害が毎回同じ質(同じ強さの敵、同じ種類のピンチ)になっていないか。障害は**外的(敵・環境)→対人(仲間との衝突)→内的(自分の弱さ)**と質を変えながらエスカレートさせると立体的になる
- シーンを1つ抜いても話が成立するなら、そのシーンは「何かを変えて」いない。すべてのシーンは状況か関係か認識のどれかを変えるべき
- 読者が先を予想できすぎていないか。ジャンルのお約束(予想できる安心感)は満たしつつ、**「予想は裏切り、期待は裏切らない」**——読者が望む感情には応え、そこへ至る経路で驚かせる
