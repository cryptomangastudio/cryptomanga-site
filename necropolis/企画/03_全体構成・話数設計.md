# 全体構成・話数設計『ネクロポリス』(仮)

想定尺:WEBTOON全40話程度(1話50〜80コマ)。三幕構成を作品全体に、5〜15話単位の中アークを各部に入れ子にする。

- ミッドポイント(偽りの勝利/敗北):第20話前後 ―「体制の内側から変える」道(カインとの共闘)が一度成立しかけ、直後にオルグレンの本当の狙いが露見して崩れる(偽りの勝利型)。
- オールイズロスト:第33話前後 ― シェイの稼働期限が強制的に打ち切られ、リゼは間に合わない。師ゲインも拘束される。「規則の中で戦う」という前提が完全に破綻する。

---

## 第一幕:第1部「弔い師見習い編」(第1〜10話・全体の設定 〜25%)

目的:世界観・職業・母の謎を提示し、「規則の中の弔い」から「規則を破る弔い」への最初の一歩を踏ませる。

```
■第1話「まだ、生きている」
- 目的:読者に世界観(弔い師/亡骸/都市)と主人公の傷を1話で体感させる
- あらすじ:新米弔い師リゼが単独任務で異常個体の亡骸に遭遇。名を呼ばれ動揺する。監督官カインは規則通り即処理しようとするが、リゼは体を張って止める。処理後、母の名が記された鍵を発見する。
- 開幕:前話なし。いきなり任務の現場から始める(掴み優先)
- 山場:リゼが規則を破ってカインの前に立ちはだかる瞬間
- 引き:謎提示型(鍵の発見)
- 仕込み:母アイリスの存在、カインとの初対立、共鳴能力の基本ルール

■第2話「鍵と、禁書」
- 目的:母の謎の輪郭を見せる/師ゲインとの関係性を提示
- あらすじ:鍵の正体を探るリゼ。師ゲインに尋ねるが言葉を濁される。ギルドの資料庫で母の名が抹消されていることに気づく。
- 山場:資料庫で見つかる、母の名前だけが黒塗りにされたページ
- 引き:謎提示型(何かを知っているゲインの表情)
- 仕込み:ゲインが抱える秘密、「大弔い事件」というワードの初出

■第3話「監督官ヴォス」
- 目的:カインのWant/Needを提示し、単なる敵役ではないと分からせる
- あらすじ:リゼの規則違反が評議会に報告されかける。カインは意外にも独断で握りつぶす。理由を問うリゼに、カインは「効率を乱されただけだ」と突き放す。
- 山場:カインが部下(発電区の管理者)を庇う場面で、彼の有能さと一定の情の厚さを行動で見せる
- 引き:決意型(リゼ「――あなたのことも、知りたい」ではなく、鍵の謎を自分で調べる決意を宣言)
- 仕込み:カインの家柄(評議会御曹司)が判明

■第4話「発電区」
- 目的:都市の階層構造(格差)を可視化する
- あらすじ:鍵に導かれ発電区へ潜入するリゼ。亡骸たちが稼働する現場の過酷さを初めて目の当たりにする。
- 山場:整然と並んで働く無数の亡骸の見開き(WEBTOONでは大きな縦スクロール1画面)
- 引き:反転型(亡骸の一体が、明らかに"自我"を持ってリゼの名を呼ぶ ―― シェイ登場の予兆)
- 仕込み:発電区の生活環境、シェイの初登場

■第5話「シェイ」
- 目的:シェイとの関係を立ち上げる。テーマ(弔われる側の視点)を導入
- あらすじ:自我を保った亡骸シェイと言葉を交わすリゼ。シェイは「稼働をやめさせられるのが怖い」と本音を漏らす。
- 山場:シェイが生前の記憶(誰かに覚えていてほしいという願い)を語る静かな場面(セリフを絞り、絵の間で見せる)
- 引き:感情型(シェイの言葉に、リゼが何も返せない)
- 仕込み:「弔い=終わらせること」への疑問の種

■第6話「規則という祈り」
- 目的:リゼの内面の葛藤を深める。ギルドの"建前"を体感させる
- あらすじ:ギルド内の弔いの儀式に参加するリゼ。表向きの美しさと、裏の実態(稼働期間の管理)のギャップに気づき始める。
- 山場:儀式の美しい演出と、直後に見える"処理"の事務的な現実の対比コマ
- 引き:謎提示型(儀式で使われる詠唱の一節が、母の日記の文言と一致する)
- 仕込み:母アイリスが儀式の設計に関わっていた可能性

■第7話「鍵の中身」
- 目的:第一幕の中間の小さな決着(鍵の謎の一部解明)
- あらすじ:鍵が開けるのはギルド地下の私物保管庫。そこにアイリスの日記の一部を発見する。日記には「完全弔い」という言葉が記されている。
- 山場:日記の一節が明かされるコマ(初めての「完全弔い」という単語の提示)
- 引き:謎提示型(日記の続きが破り取られている)
- 仕込み:完全弔いという禁忌ワードの初出(世界観の核心に触れ始める)

■第8話「対立」
- 目的:カインとの対立を明確な形にする
- あらすじ:リゼの独自調査がカインの知るところとなる。カインは「これ以上は君のためにならない」と警告するが、リゼは引かない。
- 山場:互いの価値観が真正面からぶつかる会話劇(バトルではなく対話で緊張を作る)
- 引き:危機型(カインが評議会上層(オルグレン)に報告するかどうかの選択を迫られる)
- 仕込み:カインとオルグレンの血縁関係が明かされる

■第9話「シェイの期限」
- 目的:テーマを行動レベルの緊急性に変える
- あらすじ:シェイの稼働期限(規定の猶予期間終了)が迫っていることが判明。リゼは規則上の"処理"を任される立場に置かれる。
- 山場:リゼが担当を外してほしいと師ゲインに懇願する場面
- 引き:決意型(リゼ「わたしが、ちゃんと弔う」)
- 仕込み:第二幕への転換点(第一幕の"知る"フェーズから"動く"フェーズへ)

■第10話「初めての選択」
- 目的:第一幕のクライマックス。Point of No Return
- あらすじ:リゼは規則の期限より前に、シェイを"完全に"弔うという禁忌に踏み込む決断をする(実行はまだ)。ゲインがそれを黙認する。
- 山場:リゼが完全弔いの技法を初めて学ぶ、緊張感のある師弟のコマ
- 引き:決意型(次話、実行に踏み切る)
- 仕込み:読者に「これは規則を破る物語だ」と明確に宣言する回
```

## 第二幕:第2部「発電区編」(第11〜25話・対立 〜50%)

- 第11〜13話:シェイへの完全弔いの実行と、その代償(過共鳴症の初兆候)。カインが目撃し、報告するか迷う。
- 第14〜17話:カインが黙認する代わりに「体制の内側から変える」共闘を提案。二人の一時的な協力関係(力関係が固定されないよう、互いに助け合う場面を作る)。
- 第18〜20話:**ミッドポイント(偽りの勝利)**。共闘によって発電区の労働環境改善案が評議会で可決されかける。しかしそれはオルグレンが「ガス抜き」として最初から仕込んでいた計画だったと判明。
- 第21〜23話:師ゲインが真相の一部(母アイリスは"暴走"したのではなく、完全弔いのシステムを都市全体に告発しようとしていた)を語る。
- 第24〜25話:オルグレンがリゼたちの動きを察知し、包囲網を狭める。ゲインが拘束される。

## 第三幕:第3部「屍都の灯編」(第26〜40話・解決 〜25%)

- 第26〜29話:リゼとカインが対立から再び袂を分かつ(カインは「体制の中での改善」を諦めきれない)。
- 第30〜32話:母アイリスの日記の全貌が判明。10年前、アイリスは告発の証拠を都市中枢に隠して処刑された。
- 第33話:**オールイズロスト**。シェイの稼働期限が強制打ち切りされ、リゼは間に合わない。ゲインも拘束されたまま。リゼは独りで最大の禁忌(都市規模の完全弔いの発動=発電網の一部停止)に向き合うことになる。
- 第34〜37話:リゼの決断と実行。カインが土壇場で協力に戻る(彼のアークの到達点:効率と存続以外の価値観で人と繋がる選択)。
- 第38話:オルグレンとの最終対峙。彼にも一定の正しさがあったことを認めた上で、それでもリゼは違う答えを選ぶ(勝敗ではなく価値観の選択で決着)。
- 第39〜40話:都市の一部の灯が落ちる代わりに、シェイをはじめとした死者たちが本当に解放される。都市がこれからどう変わっていくかの萌芽を見せて幕。ラスト1〜2話は余韻(母の墓前でのリゼの独白など)に使う。
