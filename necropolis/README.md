# ネクロポリス(仮)― 漫画化を見据えたライトノベル企画

CryptoManga Studios のオリジナルIP企画。死者の魂を動力に変えて存続する地下都市を舞台にしたダークファンタジー。Web小説として投稿し、実績を作りながらコミカライズ(漫画化)を狙う前提で設計している。

> タイトル・キャラクター名はすべて仮称。企画書 v0.1 の段階。

---

## まず読むもの

1. `企画/00_企画書.md` ― 1〜2枚のログライン・企画意図・差別化(**最初にこれだけ読めば全体像が分かる**)
2. `docs/持ち込み・漫画化戦略.md` ― どう投稿し、どう漫画化に繋げるかのロードマップ
3. `tracker/進行管理.md` ― 現在の進捗とユーザー確認待ちの項目一覧

## 企画の骨子(1行で)

死者を働かせ続けなければ灯が消える都市で、新米弔い師の少女が「弔いを規則の遂行から本当の解放へ取り戻す」ために、体制の禁忌に踏み込んでいく話。

## 現状とネクストアクション

企画書・世界観・キャラクター・全体構成(第1部は話数詳細まで)・あらすじ集・第1話ネーム構成案・漫画化戦略まで一式そろえた「たたき台(v0.1)」の段階。タイトルやキャラ名の確定、投稿プラットフォームの選定など、進める前にユーザー確認が必要な分岐点は `tracker/進行管理.md` の「次の意思決定待ち」にまとめてある。次に会話を再開したら、そこから確認していく想定。
