# ショートドラマ原作事業(第0段階: 実測)

CryptoManga Studios の縦型ショートドラマ/ショートアニメ向け**原作供給事業**の土台。「AIオッケーな小説プラットフォーム」構想の徹底検証から出発し、「プラットフォームは作らない。原作スタジオとして既存の買い手に売る」に組み替えた(経緯は `docs/research/`)。

> このフォルダも kindle/ と同じ流儀: 煽らず、検証結果に基づき、できること・できないことを正直に置く。

---

## 事業の結論(徹底リサーチの回答)

- **「AI許容の汎用投稿プラットフォーム」= 勝ち筋なし**(AI許容は既に業界標準の開示運用/中国で品質反発の実証/後発投稿サイトは大資本でも全滅の歴史)
- **「縦型原作の供給」= 条件付きで勝ち筋あり**。需要は実在(日本は世界2位の高課金市場で原作調達に各社奔走)、かつ**卡点・フック・反転の「短劇の技術」は日本で供給不足**(中国側実務者の評価)——この技術＋当社の編集力が差別化
- **AI補助+人間編集+証跡管理は法務上の参入資格**(文化庁「創作的寄与」要件)。完全AI生成は著作権が立たず買い手がつかない=AI量産勢は入れない。kindle事業の運用(`kindle/templates/AI利用記録.md`)をそのまま流用
- 段階設計: **第0段階(投資ほぼゼロ)=拉片→制作→既存窓口に応募して実測** → 第1段階=当たった型のレーベル化 → 第2段階=引き合いが実在した場合のみ場の構築を再検討
- **停止ライン: 6カ月で採用・受賞・オファー0件なら撤退**(出口をコミカライズ本業に戻す)

## 第0段階の実行計画

1. **拉片(いまここ)**: BUMP上位3作を秒単位分解し「日本の現地係数」を実測 → `拉片/拉片の手引き.md`
2. **SSFF企画コンテスト応募**(締切2026/8/9・製作資金1,000万円・法人可・生成AI申告制)
3. **BUMP仕様の原作制作**(1話1,800〜2,000字×20話、不倫・復讐/ミステリー/恋愛)→ Nola掲示板+BUMP for CREATORS持ち込み
4. **Minto業務委託応募**(即金+審査フィードバック=無料の添削)
5. 進捗と応募結果は `tracker/応募トラッカー.md` に記録

├── docs/
│   ├── 2026-07_縦型原作 応募窓口一覧.md      ← 応募先23窓口(締切・報酬・権利・応募順)
