## 主人公 ── 早川真紀(はやかわ まき)/ 30歳 / 介添人

### 一行でいうと
「他人の式を九年、世界で一番きれいに看取ってきた女が、自分の式だけは挙げていない」。

### 双層身分(このジャンルの背骨)
- **表**: どんな花嫁の一日も完璧に整える、透明な裏方。目立たず、決して主役にならない。
- **裏**: 式場の全裏側(控室・進行表・花嫁の手紙・封筒)に合法的に立ち会える唯一の他人であり、人と部屋を"読む"観察眼と鉄の自制心を持つ。**その職能こそが、内側から式を解体できる武器**。
- ※「隠された財閥令嬢」型ではない。彼女のチート(金手指)は超常能力ではなく**「介添人という職業のアクセス権＋観察力＋崩れない敬語」**。地に足のついた双層身分。

### Want / Need / Wound
- **Want(表の目的)**: 夫・亮太と梨花子に、相応の代償を払わせる。
- **Need(本当に必要なもの)**: 他人の幸せの脇役でいることをやめ、自分の人生を主役に取り戻す。挙げられなかった自分の式を、自分のために弔う。
- **Wound(欠落の原点)**: 二段構え。①直接の傷=夫の「式に何十万もかけるの、もったいなくない?」の一言で、式のプロなのに自分の式を写真だけにした。②より深い傷=真紀は昔から「誰かの晴れ舞台を支える側」に回り続けてきた人間(長女気質。妹の受験・親の記念日・友人の結婚、いつも準備する側で、自分が主役の日を持たないまま来た)。だから介添人という「一生、他人の主役の日を作る仕事」を天職と思い込んでいる。

### 記憶点(瞬時に彼女と分かる印)
1. **手帳の正の字**: 担当した式で花嫁が泣いたら一画足す。三百二十一。意味のない癖(=ノイズ。20話で一度だけ効かせる)。※本編中に自分では説明しない。
2. **物を寸分違わず戻す癖**: パンフレットの角度、封筒の向き。九年で身についた「原状回復」の職業病。緊張・動揺時ほど出る。
3. **崩れない敬語**: 怒りや動揺は、口調が乱れるのではなく**より完璧な敬語になる**ことで表れる。

### 爽点功能(この人物がどの快感を担うか)
- **冷静な実力碾圧・原話奉還を敬語で放つ**エンジン。声を荒げず、微笑んだまま相手の言葉をそのまま返す。「取り乱す被害者」ではなく「一分の隙もない仕事人」であることが爽。

### 声の仕様(全話で固定 ── ここがブレるとキャラが溶ける)
- 一人称: **「私(わたし)」**。
- 地の文: 短文主体。**観察・計測の視点**(数える、角度を見る、温度を測る、匂いを識別する)。感情を直接ラベリングせず、身体反応と細部で見せる。
- セリフ: 客・同僚には完全な敬語。**動揺しても敬語を崩さない**(崩れるのは、20話でただ一度だけ)。
- モノローグ: 皮肉を自分で解説しない。読者に落とさせる。
- ✕やらせない: 感情的な絶叫、長い自己憐憫の独白、涙を人前で見せる。

### 禁則(絶対にしないこと=キャラの魂)
- **式そのものを壊さない。花嫁の一日を台無しにしない。** 復讐は「式後の人生」への社会的帰結で果たす。破壊ではなく完璧さで復讐する。
- 人前で泣かない/取り乱さない。
- ※この禁則が19話で一度だけ試される(下記アーク参照)。禁則があるからこそ、それを賭ける瞬間が効く。

### 得意 / 致命的に苦手
- 得意: 人と場を読むこと。段取り。感情の制御。
- 苦手: 自分のために何かを望むこと。「あなたはどうしたいの」と聞かれること。

### アーク(フラット型 ── ほぼ変わらない。変化は1点のみ)
- 核・信念・有能さは**最初から最後まで固定**。世界(周囲)が彼女の真価に気づいていく=フラットアーク。
- **唯一の変化ビート(19〜20話)**: 「すべての式を守らねば」という強迫を、一度だけ手放す。梨花子を庇うのをやめ、手紙を渡す。そして自分の式を自分のために取り戻す。→ これは"じわ成長"ではなく、**打点の決まった一手**。

### テーマ的役割
「支える側/支えられる側」の固定した力関係を、自分の意志で一度だけ動かす人。

## 大反派 ── 白石梨花子(しらいし りかこ)/ 27歳 / 新婦

> **方針(著者決定 2026-07-21)**: 純粋に憎ませる。同情を誘うWoundは持たせない。「可哀想な過去がある悪役」という予定調和を、逆に**"同情話は本人の嘘"というどんでん返しの燃料**として使う。

### 一行でいうと
無邪気の仮面で「完璧な女」を選び、その人生を内側から壊すことに悦びを覚える、連続的な捕食者。

### 双層身分
- **表**: 甘えん坊で無邪気、少しわがままな"かわいい"花嫁。過去に傷を負った可哀想な女、という顔も**演じる**。
- **裏**: すべてを計算し、標的を数ヶ月かけて調べ上げ、完璧な仮面で近づく捕食者。**真紀が亮太の妻だと最初から知った上で、亮太を誘惑して近づき、真紀を指名した。亮太は目的ではなく"真紀に一番効く壊し方"のための入口(餌)**(=偶然の出会いではない。全話厳守)。

### Want / Need(=同情の余地を作らない)
- **Want**: 「完璧に幸せそうな女」を選んで、その人生を内側から崩し、崩れる顔を一番近くで見ること。今回の獲物は真紀。玉の輿の式も夫の略奪も、真紀を壊すための手段。
- **Need**: **なし。** 彼女は欠落を埋めようとしていない(ここが"純粋な悪役"の肝)。強いて言えば、他人の幸福が崩れる瞬間だけが快楽。
- **可信の担保(同情ではなく恐怖で)**: 生い立ちの悲劇や裏切られた過去は**用意しない**。彼女は「そういう人間」。ただし知能・忍耐・演技は本物で、この"手際の良さ"だけが「なぜ今まで見抜かれなかったか」に答える。

### 偽の同情バックストーリー(=予定調和を裏返す仕掛け)
- 表向き「二年前、婚約者の不倫で婚約破棄され、衆前で恥をかいた被害者」という物語を、母や自分の口から流す(4話・7話で提示)。真紀も観客も一度ほだされる。
- **真相(中盤の反転・18話)**: それは嘘。前の婚約は**梨花子自身が略奪・崩壊させたもの**で、彼女は加害者だった。さらに二年前、別の式場で「担当の介添人が泣いて壊れて辞めた」——それも梨花子の仕業(3話の噂の正体=前の犠牲者・桐谷奈津)。**真紀は最初の獲物ではない。**
- → この事実は同情ではなく「次は真紀」という恐怖と、「同情話に騙されていた」という怒りを生む。

### 記憶点
1. 口癖: **「よろしくね、介添人さん」**(役職で呼ぶ=真紀を"個人"として見ない冷たさ)。
2. 無邪気な笑顔が、計算には決して届かない。
3. 人との距離を詰める(身を寄せる、手を握る)。物理的な侵入で相手を支配する。

### 爽点功能(悪役として)
- **可恨**: 笑顔のまま、同情話まで武器にして標的を嬲る。読者の怒りを最大まで溜める装置。
- **可信**: 悲しい過去ではなく、"手際の完璧さ"と連続性(前の犠牲者の存在)で筋を通す。
- **递进**: 無邪気の皮(1〜7)→ 確信犯と判明(8)→ 偽の同情が崩れ捕食者と判明(18)→ 決戦で自分の反復パターンに足を取られる(28)。

### 声の仕様(全話固定)
- 一人称: **「私(わたし)」**だが、甘え口調。語尾がやわらかい。
- 公の場: 徹底して"かわいい"仮面。敬語も可愛く崩す。
- 真紀と二人きり: 仮面を外し、静かで確信に満ちた低い温度になる(8話の鏡のシーンが初開示)。
- ✕やらせない: 公の場でのマスク剥がれ(=素顔は真紀の前でしか出さない。27話で公衆の面前で崩れるのが唯一の例外=最大の爽点)。

### 禁則
- 公衆の面前では、絶対に"かわいい"仮面を落とさない(唯一の例外=27〜28話の敗北)。
- 自分の本性(捕食の悦び)を、真紀と二人きりの時以外に見せない。

### アーク(ネガティブ型)
- 変わらず堕ちていく。**自分が繰り返してきた同じ手口(同じ偽の同情話・同じ略奪のMO)を、記録魔の真紀に照合され、衆前で立証されて自壊する**=自業自得の完成。同情ゼロで終わらせる。

## 前の犠牲者/鍵の証人 ── 桐谷奈津(きりたに なつ)/ 34歳 / 元・介添人

### 機能
- **二年前、梨花子に同じ手口で人生を壊された前の獲物**(3話「別の式場で担当が泣いて辞めた噂」の正体)。真紀にとって「このまま行けば自分がこうなる」という**壊れた未来の鏡**(彩香=過去の自分、奈津=あり得た未来、の対構造)。
- 中盤で真紀が捜し当てる(15〜17話)。今は式場を辞め、心を病んで人前に出られない。最初は証言を怖がるが、決戦で梨花子の連続犯を立証する鍵の証人になる(28話)。
- **梨花子が"連続犯"である物証・証言の担い手**。彼女がいるから、決戦は「一件の不倫の暴露」ではなく「捕食者の連続犯の立証」に格上げされ、爽が跳ねる。

### 造形(中密度)
- かつては真紀のように有能な介添人だった。今は抜け殻。ぽつぽつとしか話せない。
- Want: 忘れて、静かに暮らしたい。/ 変化: 恐怖を越えて一度だけ前に出る(真紀の19〜20話の一手と響き合う脇の山)。
- 一人称「わたし」。言い淀み・沈黙が多い。回復しきらないまま終わる(安易に救わない=リアリティ)。

## 中反派 ── 早川亮太(はやかわ りょうた)/ 32歳 / 真紀の夫・住宅設備メーカー営業

### 一行でいうと
「悪人ですらない。ただ、自分がいちばん可愛い男」。

### 双層身分
- **表**: 穏やかで角のない、優しげな夫。
- **裏**: 本体は**保身**。愛情でも欲望でもなく「自分が損をしない・責められない」が全行動の芯。**そもそも梨花子に"真紀へ近づく餌"として狙って誘惑された側**で、本人はそれに気づかず本気になっていく(=彼もまた梨花子に使い捨てられる駒。これが梨花子の憎さを増し、亮太を"黒幕"にしない)。式を止めに動くのも「巻き込まれて自分が破滅するのが怖い」から。

### Want / Need / Wound
- **Want**: 波風を立てずに、都合よく両方(妻の生活基盤と梨花子)を持ち続けたい。
- **Need**: 自分の選択に責任を取ること(最後まで取れない)。
- **Wound**: 明示的な悲劇はなし。**あえてWoundを薄く設計**(=「深い事情のある魅力的な悪役」にしない。真紀が三年耐えた相手が"ただの器の小さい男"だった、という虚しさこそがテーマに効く)。

### 記憶点
- 都合が悪くなると「いや、そうじゃなくて」と前置きして言い訳を始める。
- 真紀へのLINEはいつも短く、絵文字ひとつ(誠実そうに見せる最小コスト)。

### 声の仕様 / 禁則
- 一人称「俺」。柔らかい口調だが、追い込まれると早口の言い訳になる。
- **禁則**: 自分から真実を打ち明けない。決定的な場面で必ず逃げる/他人のせいにする。
- ✕やらせない: 潔い謝罪、自己犠牲。最後まで保身。

### 爽点功能 / アーク
- 主人公が**一度だけ負ける相手**(13話・「私怨での接触」クレームを先回りで入れ、真紀を担当から外させる=どん底の作り手)。ネガティブアーク。最後は離婚届を差し出され、言い訳したまま退場。

## 反転役(協力者) ── 西岡誠(にしおか まこと)/ 38歳 / 新郎・婚活アプリ運営会社役員

### 一行でいうと
「将棋指しのように、盤面を最後まで読んでから駒を動かす男」。

### ★恋愛関係の確定事項(結末ロック)
- **西岡は最後まで"協力者"であり、真紀の新たな恋愛相手にはしない**(著者決定 2026-07-21)。
- 二人の関係は「同じ痛みを知る者どうしの、静かな共闘」に留める。ラストで淡い恋愛の予感も残さない。真紀は誰とも結ばれず、**自分の式を自分のために取り戻して終わる**。これが本作の芯(他人の脇役だった女が主役になる)を最も濁らせない。
- → 西岡の全セリフ・視線から、真紀への恋愛的関心を匂わせる描写を排除する。

### Want / Need / Wound
- **Want**: 妻・梨花子の裏切りを、自分の式という最も相応しい場で、自分の手で裁く。
- **Need**: 「選ばれる/裏切られる」を勝敗で考えるのをやめること(梨花子とは別の形で、彼もこのゲームの内側にいる)。
- **Wound**: 梨花子の不倫を早い段階で察知していた。婚活アプリを運営する立場ゆえ「人が人を選ぶ仕組み」を知り尽くし、そのうえで裏切られた皮肉。

### 記憶点 / 声の仕様 / 禁則
- 趣味=将棋。会話で先回りする(相手の次の一手を読んだ返答をする)。
- 一人称「私」。低く落ち着いた声。感情を表に出さない。
- **禁則**: 取り乱さない。感情的な報復をしない(あくまで盤面で詰める)。

### 爽点功能 / アーク
- **局中局の担い手**。真紀の禁則(式を守る)を動かす鍵。フラットアーク(彼は変わらず、真実を淡々と盤上に並べる)。
- 6話で初登場(「妻になる人の全部を知りたい」)、10話で正体開示(「私の式は、彼女の刑場です」)、19話で暴露を実行。

## 準・小反派/脇役(密度は意図的に軽く ── 機能に絞る)

### 持田彩香(もちだ あやか)/ 24歳 / 後輩介添人
- **機能**: 「私になるところだった」共感の鏡。真紀を慕い、現場で学びたくて担当替えを断る(4話)。だが善意で真紀の情報を漏らす小さな裏切り(7話・11話→14話で露見)。
- **禁則**: 悪意では動かない(裏切りは全て善意・未熟さから)。だから真紀は彼女を責めきれない。
- 一人称「私」。素直で少し早口。

### 久保田(くぼた)/ 46歳 / 先輩キャプテン
- **機能**: 序盤の職場の壁(小反派)。真紀の指名客を自分に回したがる。4話で「日程調整のミス」を口実に担当替えを画策するが、真紀の正確な記録の前に空振り(小さな打脸)。以降フェードアウト。
- 深い設定は不要(序盤の腕試しの的)。

## この聖書の使い方(執筆ルール)

1. 各話を書く前に、登場人物の「声の仕様」「禁則」を読み返す。
2. 迷ったら**禁則を優先**する(プロットの都合でキャラを曲げない。曲げたくなったらプロットを直す)。
3. 真紀の敬語・観察視点、梨花子の仮面/素顔の使い分けは**全話で不変**。
4. アーク上の「変化」は真紀の19〜20話の一手だけ。それ以外の"成長"を書き込まない。
