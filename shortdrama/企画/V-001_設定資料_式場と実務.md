# V-001 設定資料 ── 舞台(式場)と実務ディテール

> 目的: 実務のリアリティが伏線と決戦場を兼ねる作品なので、土台を先に固める。ここがブレると「越権」「手紙すり替えが成立しない」等の穴が全話に波及する。
> 版: v0(著者確認・必要なら取材で確定)

---

## 1. 舞台

- **式場**: 都市部の**ゲストハウス型ウェディング施設**「(仮)ヴェルデ白金」。一棟貸し切り型で、打ち合わせルーム・ブライズルーム(控室)・チャペル・披露宴会場・裏動線(バックヤード)が一施設内で完結。
  - → **撮影上の利点**: ロケ実質1箇所。縦型の量産体制(少人数・同時撮影)に合う。真紀の自宅マンションを足して**ロケ2系統**で全話が回る。
- **時代**: 現代(2026年前後)。古装・時代劇要素なし(日本の縦型では不発ジャンル)。

## 2. 介添人(かいぞえにん)の業務範囲 ── ここを最初に定義してブレを止める

一般に「介添人=挙式当日の花嫁付き添い」で、打ち合わせは**プランナーの専任**。第1話の初稿はここを混同して越権が生じた。本作では次のように**世界観として明示的に設定**する(=フィクションとしての設計。式場により実態は異なる旨を応募時に注記):

- ヴェルデ白金は**「トップ介添人の指名制」**を敷く式場。指名された介添人は、当日だけでなく**打ち合わせにも同席**し、花嫁の要望の機微を当日運営に反映する——という高付加価値サービスを売りにしている。真紀はこの式場で**指名数一位**。
- ただし**日程調整・見積・契約・生地選定はプランナー/スタイリストの領分**。介添人が越境しない。真紀ができるのは「気を利かせて場を繋ぐ」「当日運営に関わる提案」まで。
  - → 全話で厳守: 真紀に**プランナー業務を代行させない**。日程・金額・契約の話は必ずプランナーを画面に置く。

### 介添人が"合法的に"触れられるもの(=復讐の道具になる職能)
- ブライズルーム(控室)への出入り(花嫁の素顔を見る)
- 当日の**進行表(タイムライン)管理**とマイクの受け渡し
- **「花嫁の手紙」など進行アイテムの管理・手渡し**
- 来館者の案内動線・裏動線の把握
- ※これらは真紀の「双層身分(裏=式を内側から動かせる)」の具体。取材で裏取りできれば強度が増す。

## 3. 披露宴 終盤の進行(=決戦の舞台装置)

标准的な流れ:

1. …歓談・余興…
2. **花嫁の手紙**(新婦が両親への手紙を朗読)
3. **両親への花束・記念品贈呈**
4. **新郎謝辞**(新郎がゲストへ挨拶)
5. お開き・お見送り

- 介添人(真紀)は**この進行のタイムキーパーであり、手紙・マイクの受け渡し役**。
- → **結末の暴露は、この「新郎謝辞」の枠を西岡が使う**形にする(下記)。

## 4. 結末の「暴露」導線 ── 穴のない設計

初稿の「封筒がすり替わり、花嫁が自分の不倫を読み上げる」は破綻する(本人が朗読なら見て止める)。**成立する版**に置き換える:

- **仕込み(伏線)**: 西岡(新郎/婚活アプリ運営役員・将棋=先を読む男)は、**自力で妻の不倫の証拠**を掴んでいる(真紀の証言に依存しない=真紀は犯罪的な盗聴等をしない)。西岡は「自分の結婚式を、妻を裁く場にする」と決めている(ep6・ep10で真紀に接触)。
- **真紀の役割=庇うのをやめること**: 真紀の禁則は「式を壊さない/花嫁の一日を守る」。彼女は当日、**進行を完璧に回しながら、ただ一度だけ、西岡に"新郎謝辞"のマイクと間を渡す**。梨花子を庇う介入を、しない。これが真紀の唯一の禁則破り(アークの一手)。
- **暴露の実行者は西岡**: 新郎謝辞の枠で、西岡が静かに真実を述べる(梨花子自身の言葉/証拠を提示)。**自分の花嫁を、自分の式で、自分の持ち時間で裁く**——被害者が加害者を、衆前で。合规(私刑を法的・社会的帰結に留め、暴力なし。復讐が悪役より残忍にならない)を満たす。
- **梨花子の自壊**: 公の場で初めて"かわいい仮面"が剥がれる(=彼女の禁則が破れる瞬間=最大の爽点)。婚姻は破談。
- **亮太(真紀の夫)の清算**: 同じ暴露の中で不倫相手として名が出る/映る。真紀は式場を出た亮太に、用意していた離婚届を渡す。**両方の不倫者が同じ一日で代償を払う。**
- **真紀の着地**: 手紙・封筒という"真紀の職能の象徴"は、最後まで真紀の手の中にある。彼女は式を壊したのではなく、**守るのをやめた**だけ。だから禁則違反ではあっても、介添人としての誇りは保たれる(この微妙な一線が作品の品位)。

## 5. 取材が要る点 / フィクション設計で割り切る点

| 項目 | 扱い |
|---|---|
| 介添人の指名制・打ち合わせ同席 | **フィクション設計**(式場により異なる旨を注記)。取材できれば実在の式場例で補強 |
| 「花嫁の手紙」「新郎謝辞」の進行順・マイク運用 | 一般的な流れ。取材で細部(誰が手紙を預かるか)を確定できると強い |
| 婚活アプリ運営役員が証拠を掴む現実性 | フィクション設計。ただし「探偵に依頼」等の現実的手段に置き換え可 |
| 離婚届・不貞の法的扱い | フィクションの範囲。金額・慰謝料の具体を出す場合のみ要確認 |

→ 応募段階では**「実務はフィクションとして設計。式場運用は作品世界の設定」と一言注記**すれば足りる。当たって映像化が具体化した段階でブライダル実務の監修を入れる。

## 6. 「介添人」の表記方針(読みやすさ対策)

- **懸念**: 「介添人(かいぞえにん)」は一般認知が高くない。BUMP(スマホ・カジュアル層)では読みの壁になり得る。
- **方針**: 職業(題材)は変えない。以下で読みやすさを担保する。
  1. **本文の初出に必ずルビ**: 介添人(かいぞえにん)。
  2. 初出時に一行の平易な説明を添える(既にep1: 「花嫁に一番近くで付き添う裏方」)。
  3. カジュアルな会話や地の文では**「付き添い」**の語も併用してよい(硬さを避ける)。
  4. **タイトルは"介添人"の語に依存させない**か、依存する場合はルビ必須。

### タイトル案(著者が1つ選ぶ ── BUMP流=説明的+フック)
| # | 案 | 狙い |
|---|---|---|
| T1 | **夫を奪った女の、結婚式を任されました** | 専門語なし・フック強・"任されました"で捻れを予告。読みやすさ最優先 |
| T2 | 夫の不倫相手の"付き添い"になりました | 元案の骨を残しつつ「付き添い」で平易化 |
| T3 | 花嫁のいちばん近くに、夫の女がいる | 情感型。状況が一行で立つ |
| T4 | 夫の不倫相手の介添人(かいぞえにん)になりました | 元案+ルビ。職業名を看板にする |
| T5 | 最高の一日にします ―夫を奪った花嫁のために― | キャッチ「最高の一日」を主題化 |

→ 現時点のClaude推奨は **T1**(BUMPヒット作のタイトル文法に最も近く、読みの壁がない。作中は介添人+ルビで職業を立てる)。
