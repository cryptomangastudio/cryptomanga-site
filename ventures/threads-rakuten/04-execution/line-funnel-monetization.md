# LINEファネル → 収益化 設計

> 現状：**Threads(@tsukuru.laboratory)** → 固定ポスト/プロフィール → **LINE公式（lin.ee/PHP4ANnp）** → **らくスレ無料配布**、が稼働中。
> 狙い：**LINEリストを"BANされても残る本命資産"にする**。お金をかけない・押し売りしない・規約で死なない。
> 軸：**アフィリエイターの味方**（めんどくさいを道具で消し／詐欺から正直な情報で守る）。配信も"稼がせます"でなく"手間を消す・騙されない"を届ける。

---

## 0. なぜLINEが本命なのか

- Threadsは**BAN・表示制限**のリスクがある（楽天アフィで実際にBAN事例あり）。フォロワーは"借り物"。
- **LINEリストは自分の資産**。Threadsが止まっても連絡が取れる。だから **KPIはフォロワー数でなくLINE登録数**。
- 529フォロワー・6.9万閲覧を、**LINE登録に変換**するのがこのファネルの仕事。

## 1. ファネル全体像

```
[認知]   Threads投稿（有益:リンク=3〜5:1）＋ Instagram連携
   │  固定ポスト＆プロフィールで「らくスレを無料で配ってます→LINE」
   ▼
[獲得]   LINE公式に登録（lin.ee/PHP4ANnp）
   │
   ▼
[提供]   らくスレを渡す（＝価値を先に渡して信頼）
   │  使い方・伸びる投稿の型・楽天の小ワザを"配信"で継続提供
   ▼
[収益]   ① （後で・任意）有料の少額商品を"欲しい人にだけ"
             ・消費者ガイド（反詐欺）／らくスレPro（めんどくさい消し）
        ② （受け身）来たら受けるB2B ¥0前提
        ※自分の楽天アフィ報酬は収益に数えない（味方としてのお試しのみ）
```

## 2. LINE登録直後の"あいさつ〜数通"設計（自動）

押し売りせず、**価値→価値→たまに提案**。全通に広告関係の明示を添える。

| 通 | 内容 | 目的 |
|---|---|---|
| 0（登録直後） | 「らくスレはこちら🌱＋使い方1分」 | 約束を即実行＝信頼 |
| 1（翌日） | 「伸びる投稿の"1行目"テンプレ5個」 | 有益・即使える |
| 2（+2日） | 「毎回リンクを貼ると表示が落ちる話（3〜5:1）」 | 失敗回避＝信頼 |
| 3（+4日） | 「売上≠報酬。騙されないための3チェック」 | 誠実さ＝差別化 |
| 4（+7日） | 「よかったら"感想"教えてください」 | 双方向・改善ネタ・②の声 |
| 以降 | 週1ペースで①②③④の柱から配信。**提案は数通に1回だけ** | 関係維持 |

## 3. 収益化の階段（このリストで／お金をかけない順）

1. **らくスレ無料**（集客・習慣化）… 売らない。使ってもらう。
2. **少額の有料商品（後回し・任意）**：需要が見えてから、**1つだけ**。候補：
   - **消費者ガイド ¥2,980**（132名分析の匿名化・集計版。"騙されない"側の商品）
   - or **らくスレPro 買い切り ¥4,980**（履歴/一括/テンプレ拡張。**買い切り＝サブスク解約規制を回避**）
   - ※どちらも**まず10本のプライステスト**で反応を見てから。売れなければやめる。
3. **受け身B2B ¥0前提**：LINE経由で企業から問い合わせが来たら受ける（営業しない・詳細は事業計画 付録B）。

> 自分の楽天アフィ報酬は**収益の階段に入れない**（味方としての見本／お試しのみ・実額訴求しない）。

> **順番が命**：先に売らない。無料で価値→信頼→"欲しい人にだけ"少額。押し売りは529人の小さなリストを一瞬で焼く。

## 4. 守ること（規約・法務）

- **ステマ規制**：Threads投稿は #PR、LINE配信でアフィリンクを出す時も「広告/PRを含む」旨を明示。
- **特商法**：有料商品を売る段階で、販売ページに特商法表記（法人名・連絡先・価格・返金＝デジタルは返品不可を明記）。**無料配布の段階では不要**。
- **楽天規約**：らくスレは手動コピペ・自動投稿なし。反復投稿させない。
- **誇大禁止**：LINEでも「必ず稼げる/1ヶ月で月◯万」は言わない。あなたの立ち位置（"めんどくさいを消す"／誠実）を守るほど強い。
- **LINEの規約**：友だち登録の誘導・配信頻度はLINE公式アカウントのガイドラインに沿う（過度な営業配信でブロック増を避ける）。

## 5. KPI（見る順）

| 指標 | 意味 | 目安（余暇運用） |
|---|---|---|
| **LINE登録数** | 本命資産の増加 | じわり右肩上がりを確認 |
| 登録率（プロフ/固定ポスト経由のクリック→登録） | 導線の質 | 文言ABテストで改善 |
| 配信の開封・クリック | リストの温度 | 下がったら"売りすぎ"のサイン |
| らくスレ利用・感想数 | ファン化 | ②Build in Publicのネタに |
| ブロック率 | 焼いてないか | 上がったら提案頻度を下げる |
| （後）有料転換 | 少額商品の需要 | 10本テストで判断 |

## 6. まず今週やること（ゼロ円）

1. **固定ポストの文言をABテスト**（「らくスレ無料で配ってます→LINE」の1行目を2〜3案）。
2. **LINE登録直後メッセージ**を上の「0通目」に整える（らくスレを即渡す）。
3. **プロフィール**にLINE導線が1タップで届くか確認。
4. Threadsは①④（リンクなし）中心で回し、**5投稿に1回だけ**らくスレ/LINEへ。
