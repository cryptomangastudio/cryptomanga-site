# 90日アクションプラン（リーン×余暇モード・お金をかけない標準版）

> 事業計画v3の **リーン運用（年約¥57,000）を、さらに「基本は無料ツールだけ」で回す** 前提の実行手順。
> **大方針**: お金をかけない。**余暇（夜・週末）の時間だけ**で回し、本業（漫画/IP）の時間は削らない（同時稼働≤2・機会費用≒¥0）。
> 収益は追わない。まず「Cメディアの複利の種まき」と「A自社運用の一次データ蓄積」に集中する。

## 1. 時間の使い方（余暇キャップ）

- **週5〜7時間まで**（例: 平日夜30分×3＋週末2〜3時間）。これを超える月は新しいことに手を出さない。
- 同時に進めるのは**最大2つ**（このプランでは基本 **C（記事）＋A（自社運用）** の2本だけ）。
- ツール（B1/B2）・ガイド（B3）は90日時点では**まだ作らない/売らない**（種まき優先）。

## 2. 90日でやること（週割り）

### Phase 0（Week 1-2）: 土台（無料範囲）
- ニッチ確定：**ふるさと納税×食品**（旗艦）。理由＝高AOV・年末スパイク・Aの実購入と直結。
- 無料ホスティングでサイトを1枚公開（トップ＋プロフィール）。GA4/Search Console設置。
- **無料でできる法務最低線**：各ページに「アフィリエイト広告を利用しています」を常時表示（ステマ規制）。特商法表記・私書箱は「売り始める時」まで保留。
- A（自社運用・別ブランド）：楽天アフィリ/ふるさと納税アカウント準備。**自分で1件、少額の返礼品を実際に申し込む**（＝一次データの素）。

### Phase 1（Week 3-8）: ロングテール記事を積む
- **週1〜2本**、Aの実購入・実写真・実数値で肉付けした比較/レビュー記事を公開（AIは下書きのみ、必ず自分の一次情報を上乗せ）。
- 狙うのは**ヘッドKWではなくロングテール**（第4章の候補）。
- 記事は affiforge（自社のリンクボックス/比較表/ランキング）で作り込む。
- Threadsにも**手動で**三段構成の投稿（rakuaffi下書き＋#PR・手動コピペ）。ただし集客の主軸はSNSでなく**記事のSEO**に置く（Threads反復投稿のBANリスク回避）。

### Phase 2（Week 9-12）: 検証と判断
- 累計10〜15本を目標。Search Consoleで**表示回数が付き始めたKW**を確認（＝当たりの芽）。
- 当たった型・ジャンルに次の記事を寄せる。
- **90日レビュー**：オーガニック流入が「ゼロ→微増」に転じたか。転じていれば継続、まったく動かなければテーマを1回だけ入れ替え。
- ここまで**現金支出ほぼゼロ**。ガイド(B3)・有料ツール(B2)は流入が育ってから。

## 3. この90日で「やらないこと」

- 有料ツール・有料テーマ・広告出稿・私書箱契約（＝お金がかかること）は保留。
- 「◯万達成」等の実績訴求・教材化（レッドライン）。
- B2B営業（受け身のみ）。
- Threadsの毎投稿リンク／自動投稿（BAN誘発）。

## 4. Cの最初のロングテール記事テーマ候補（旗艦=ふるさと納税×食品）

ヘッド（「ふるさと納税 おすすめ」等＝大手が独占）は避け、**具体×少額×悩み**で刺すロングテールから。まずSearch Consoleに拾われることを優先。

## 5. 有料に踏み込む判断（90日以降・お金を使い始める線）

- ガイドが売れる兆し → メールESP・会計ソフト等を必要に応じ追加。
