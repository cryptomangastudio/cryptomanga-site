## 3. 90日ロードマップ

### Phase 0：設計（Day 0-3）
- ニッチ確定（例：ふるさと納税×日常使いの食品）。
- アカウント設計：プロフィール1行目に「誰の・どの悩みを・どう解決するか」。顔出し不要。
- 楽天アフィリエイト登録＋Threadsを認定SNSに設定（規約要確認。ASPによりThreads直貼り可否が違う）。
- **商品化の器を先に用意**：note/Brainアカウントを開設し「実践ログ」を書き始める（＝後で教材になる素材）。

### Phase 1：型を回す（Day 4-30）— 目標：初報酬＋フォロワー1,000
- 投稿は**三段構成**（`report.md` 5章）。1日3〜5投稿、うちリンクは3〜5投稿に1回。
- **AIで量産**：`rakuaffi`（Threads×楽天モード）で投稿文の叩き台を生成 → 自分の言葉で1〜2割書き換え。
- ネタ源：楽天ランキング／お買い物マラソン／季節イベント。
- KPI：初報酬（額より"発生させる"こと）、保存・返信が付く投稿の型を特定。

### Phase 2：伸ばす（Day 31-60）— 目標：月1〜3万円＋フォロワー5,000
- 伸びた型を横展開。上位互換の"当たり投稿"をテンプレ化。
- Instagram/noteへの誘導導線を追加（Threads単体規約リスクの分散）。
- 実額を**時系列で正直に記録**（むぎ型）。これが最強の信頼コンテンツになる。

### Phase 3：商品化（Day 61-90）— 目標：月5〜10万円＋教材/ツールの初版
- 90日のログを**「AI×ニッチで0→月◯万円の実践記録」教材**に再構成（note/Brain）。
- **誇大にしない**のが逆張りの差別化（後述の落とし穴を回避した"誠実な教材"）。
- 併売：`rakuaffi`を無料集客ツールとして紹介 → 有料教材へ送客（フリー→有料の王道ラダー）。

## 4. 落とし穴（132名分析＋一次検証で判明した実リスク）

> ここは**そのまま商品の目玉コンテンツ**になる。買い手が最も知りたい"損しない"情報。

1. **売上 ≠ 成果報酬**：「売上670万」等は楽天の流通額（GMV）で、実報酬は桁違いに小さい（カピパラおじさんの実例＝売上670万に対し報酬約20万）。数字の見せ方に騙されない・自分もやらない。
2. **BAN・規約厳格化の持続性リスク**：同一内容の反復投稿禁止など規約が厳しく、実際に有名実践者が**2026年4月に楽天アフィをBAN・収益停止**した事例あり。単一プラットフォーム依存は危険 → 導線を分散（Instagram/note/ブログ）。
3. **短期×高額の即金訴求は地雷**：「1ヶ月で月20万」系は教材のセールスコピー。現実的な初速は月1〜5万。期待値を正しく設計する。
4. **毎投稿リンクは表示減**：有益:リンク=3〜5:1を死守。
