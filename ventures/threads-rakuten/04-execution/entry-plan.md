# AI × ニッチ参入プラン — Threads×楽天アフィリで空きジャンルを取る

> ⚠️ **事業計画v3との位置づけ（重要）**: 本プランは事業計画（`../03-business-plan/business-plan.md`）の **A「自社実践」＝収益ゼロ固定のデータ供給源（別ブランド）**に相当する。必ず守る2点：
> 1. **別ブランド・別主体で運用**（Aは収益ゼロ固定のデータ源。収益系C/Bと役割を分離し、将来「受け身B2B」が来ても利益相反を避ける）。
> 2. **自社実績を「稼げる証拠」として訴求しない**（「◯万達成」等のヘッドライン化・暗示表現は社内審査で禁止）。実データは内部の一次ソース／集計版B2Cコンテンツの素材としてのみ用いる。
>
> 主目的は「大きく稼ぐ」ことではなく、**規約順守下で一次データ・運用知見を蓄積**すること。楽天/Meta規約（反復投稿・自動化の禁止）を順守し、即金訴求は行わない。

> 132名分析（`../01-market-analysis/report.md`）から導いた「まだ手薄な勝ち筋」の実行プラン。
> 前提: 顔出し不要・スマホ＋AI・1日30分。

## 3. 90日ロードマップ

### Phase 0：設計（Day 0-3）
- ニッチ確定（例：ふるさと納税×日常使いの食品）。
- アカウント設計：プロフィール1行目に「誰の・どの悩みを・どう解決するか」。顔出し不要。
- 楽天アフィリエイト登録＋Threadsを認定SNSに設定（規約要確認。ASPによりThreads直貼り可否が違う）。
- **内部データ記録の器を用意**：運用ログを書き始める（＝集計版B2Cコンテンツ（B3）の内部素材。特定個人の名指しや実額ヘッドラインには使わない）。

### Phase 1：型を回す（Day 4-30）— 内部目標：初報酬の発生＋フォロワー1,000
- 投稿は**三段構成**（`../01-market-analysis/report.md` 5章）。1日3〜5投稿、うちリンクは3〜5投稿に1回。
- **下書きをAI生成**：`rakuaffi`（Threads×楽天モード）で投稿文の叩き台を生成 → 自分の言葉で1〜2割書き換え → **手動でコピペ投稿**（自動投稿しない）。
- ネタ源：楽天ランキング／お買い物マラソン／季節イベント。
- KPI：初報酬（額より"発生させる"こと）、保存・返信が付く投稿の型を特定。

### Phase 2：伸ばす（Day 31-60）— 内部目標：月1〜3万円（運用KPI・対外訴求しない）＋フォロワー5,000
- 伸びた型を横展開。上位互換の"当たり投稿"をテンプレ化。
- Instagram/noteへの誘導導線を追加（Threads単体規約リスクの分散）。
- 実額を**時系列で正直に記録**（むぎ型）。※あくまで内部記録。「◯万達成」等の対外ヘッドラインには用いない。

### Phase 3：データ蓄積と検証（Day 61-90）
> ⚠️ 事業計画v3により、**このログを「0→月◯万円」型の教材として当社が販売することはしない**（即金・再現性暗示はレッドライン）。ログは以下の用途に限定する。
- 90日のログ・実データを、**事業計画v3のB3「消費者ガイド（匿名化・集計版）」の内部素材**として蓄積（実額ヘッドライン化・再現性訴求はしない）。
- 楽天/Threads規約の運用実感（何が反復投稿判定されるか等）を、**rakuaffi のガードレール改善**にフィードバック。
- 集客導線の実測（ツール→コンテンツのCVR）を、事業計画のフェーズ1ファネル検証に接続。

## 4. 落とし穴（132名分析＋一次検証で判明した実リスク）

> ⚠️ これらの知見は**当社自身が踏まないため**の内部ガイド。商品化する場合も**匿名化・集計版**（事業計画v3 B3）に限り、特定個人の名指し評価は商品に載せない。

1. **売上 ≠ 成果報酬**：「売上670万」等は楽天の流通額（GMV）で、実報酬は桁違いに小さい（カピパラおじさんの実例＝売上670万に対し報酬約20万）。数字の見せ方に騙されない・自分もやらない。
2. **BAN・規約厳格化の持続性リスク**：同一内容の反復投稿禁止など規約が厳しく、実際に有名実践者が**2026年4月に楽天アフィをBAN・収益停止**した事例あり。単一プラットフォーム依存は危険 → 導線を分散（Instagram/note/ブログ）。
3. **短期×高額の即金訴求は地雷**：「1ヶ月で月20万」系は教材のセールスコピー。現実的な初速は月1〜5万。期待値を正しく設計する。
4. **毎投稿リンクは表示減**：有益:リンク=3〜5:1を死守。

## 6. 事業全体の中での位置づけ（v3整合）

本プランは事業計画v3の **A「自社実践（収益ゼロ固定・データ源・別ブランド）」**。全体像は下記（詳細は `../05-products/product-catalog.md`・`../03-business-plan/business-plan.md`）。

```
[主柱/メディア]  C オウンドメディア（ふるさと納税×食品 等・比較/レビュー）
      ▲ 実データ・実写真・実数値を供給
[背景/別ブランド]  A 本プラン（AI×ニッチ自社運用・収益ゼロ固定）
      │
[高粗利/BE]  B3 消費者ガイド（匿名化・集計版）  ←Aのログが内部素材
[無料/入口]  B1 rakuaffi無料版 →（条件付き）B2 買い切りPro
      （育てば副産物として）→ 受け身B2Bの問い合わせ（営業なし・¥0前提）
```

本プランの役割は**Cの一次コンテンツ素材の供給と、規約運用知見の蓄積**。実名リストの販売や「稼ぎ方マップ」の販売は**しない**（レッドライン）。自社実績を「稼げる証拠」として訴求もしない。
