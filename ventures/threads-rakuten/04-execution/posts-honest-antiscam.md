## A. 「売上 ≠ 報酬」を伝える（最重要インサイト）

### 1. 売上670万のカラクリ
```
「楽天アフィで売上670万」——この数字、そのまま鵜呑みにしないでほしい。

売上（＝流通額）と、自分の手取り（成果報酬）は別モノ。
報酬は売上の数%だから、売上670万でも手取りは20万前後、なんてことは普通にある。

嘘とまでは言わないけど、"売上"を大きく見せて"収入みたい"に錯覚させる見せ方は多い。
数字を見たら「これは売上？報酬？」って一回止まるだけで、騙されにくくなる。

同じ疑問持ったことある人、いる？
```

### 2. 数字の主語チェック
```
稼いでる系の投稿を見たとき、たった1つ確認するだけで冷静になれる。

「その金額の"主語"は何か」

・売上（流通額）→ 手取りじゃない
・成果報酬 → これが本当の手取り
・"売上"だけデカく出して報酬を言わない → 要注意

主語をぼかしてる数字ほど、盛られてる可能性が高い。
これ、覚えておくと一生使えます。保存推奨。
```

## B. 持続性リスク（BAN・規約）を伝える

### 3. 「BANされた話」
```
楽天アフィで"有名な実践者"が、ある日いきなりアカウントBAN・収益ゼロになった実例がある。

理由は規約。同じ内容の反復投稿とか、自動化っぽい動きとか、線を越えると一発。
「今月◯万でした」の裏で、翌月ゼロになるリスクは常にある。

だから1つのプラットフォームに全部乗せるのは危ない。
稼ぐ前に「消えても残る場所（メール/LINE/ブログ）」を作っておくほうが先。

これ、稼いでる人ほど言わないやつです。
```

### 4. 「毎回リンク」が地雷な理由
```
アフィリ始めたてでやりがちな失敗：毎投稿にリンクを貼る。

プラットフォームは"宣伝ばかりのアカウント"の表示を静かに下げる。
下手すると規約違反で凍結の入口。

目安は「有益な投稿：リンク＝3〜5：1」。
リンクを我慢するほど、逆にリンク投稿が届く。急がば回れ、ほんとにこれ。
```

## C. 情報商材・誇大の見分け方

### 5. 情報商材に騙されない3チェック
```
「これ買えば稼げます」系を見たとき、この3つを確認してほしい。

① 数字の主語（売上？報酬？→ ぼかしてたら赤信号）
② 再現性の断定（"誰でも""必ず""1ヶ月で"→ 盛ってる）
③ 出口（無料の親切そうな話が、最後に高額塾に繋がってないか）

3つ全部クリアしてる発信者は、正直かなり少ない。
逆に、この3つを自分から開示してる人は信用していい。
```

### 6. 「無料」の出口を見る
```
"無料"って言葉、いちばん警戒したほうがいい場面がある。

無料プレゼント → LINE登録 → 無料相談 → 高額の塾/コンサル

この導線、無料の親切がゴールじゃなくて、高額バックエンドの入口になってることがある。
無料そのものは悪くない。でも「この人の"出口"はどこ？」だけ先に見ておくと、冷静でいられる。
```

### 7. 「1ヶ月で月20万」を信じてはいけない
```
「1ヶ月で月20万」——この手のコピー、ほぼセールス用の見出しです。

現実の初速は月1〜5万がいいところ。しかも続く保証はない。
"最速・最短・誰でも"が並ぶほど、期待値だけ吊り上げて売る設計。

夢を否定したいんじゃなくて、正しい期待値でスタートしたほうが、結局続くし折れない。
遠回りに見えて、これがいちばん稼ぎに近いです。
```

## D. 誠実な現実（自分ごととして正直に）

### 8. 稼げなかった月の話
```
正直に言うと、アフィリって"ゼロの月"が普通にある。

「今月も発生1件」みたいな時期を、みんな投稿しないだけ。
表に出てくるのは上振れた月のスクショばかりだから、初心者が「自分だけダメだ」って錯覚する。

ゼロは失敗じゃなくて、ほぼ全員が通る通過点。
そこで辞めるか、型を1個直して続けるかだけの差です。
```

### 9. 味方でいたい理由（自己開示）
```
自分でガッツリ稼ぐより、"頑張ってる人が騙されないこと"のほうに関心がある。

情報商材で最初の一歩を折られる人を見るのが、正直いちばんイヤで。
だから「めんどくさいを道具で消す」と「誇大に騙されないための正直な情報」を配ってます。

稼がせます、とは言いません。
でも「損しにくくする・続けやすくする」なら、ちゃんと手伝えます。
```

### 10. フォロワー数の幻想
```
「フォロワー1万人＝稼いでる」も、実はそうとは限らない。

検証できた範囲だと、公言してる万垢の"実フォロワー"が数千だった、なんてこともある。
数字は盛れる。フォロワーも、収入も。

だから他人の数字で焦らなくていい。
見るべきは「その投稿、自分が保存したくなったか」だけ。土台はそこです。
```
