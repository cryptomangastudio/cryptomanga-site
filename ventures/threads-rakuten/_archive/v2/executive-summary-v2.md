# エグゼクティブサマリー：CryptoManga Studios LLC 事業計画 v2.0

作成日: 2026-07-07 ／ 計画期間: M0(2026-07)〜M12(2027-07) ／ 通貨: JPY(税抜・損益ベース)

## 1. 結論：条件付き限定GO
着手は**フェーズ0(コンプラ基盤+無料入口ツール)とフェーズ1(消費者保護型B2Cコンテンツ)に限定**。本命のB2B真贋監査は3ゲート全通過まで資源投入を凍結。素案の「初心者向けノウハウ販売」「実名リストの無同意販売」は棄却(レッドライン)。B2Bは「証明済みの本命」ではなく「安く検証する仮説」として扱う。

## 2. 収益源と着手順序
1. **rakuaffi(Threads×楽天モード／無料)** — ファネル入口。下書き生成+手動コピペ限定(自動投稿しない)。
2. **市場構造リテラシー・コンテンツ(B2C／3,000円)** — 早期少額キャッシュ＋消費者保護ブランド。稼ぎたい初心者ではなく「情報商材被害を避けたい層」向け。粗利約70%。
3. **B2B真贋監査(スポット受託15万円/件、モニタリング10万円/月)** — 仮説の柱。3ゲート通過後のみ着手、M8以降。粗利約75〜80%。
- 背景：自社運用アフィリ(別ブランド、利益相反回避)。

## 4. 最初の90日にやること
- **フェーズ0(0-1ヶ月)**：法務ガードレール実装(広告明示デフォルト付与／特商法表記／誇大禁止ワードブロック／匿名化)、rakuaffi の楽天・Meta規約適合の**書面確認**、決済・配信の複数系列確保。並行してB2B需要ヒアリング(想定買い手10〜15社)と個情法27条の弁護士DDに着手。
- **フェーズ1(1-3ヶ月)**：匿名化・集計版B2Cコンテンツ販売、ファネル3段(週次UU→登録率→CVR)の実弾テストでCAC≈0仮説を検証。
- **必達ゲート**：フェーズ0の3着手ゲート(法務／規約適合書面／決済複数系列)未達なら全前進を停止。

## 5. 最大リスクと対策
| リスク | 対策 |
|---|---|
| **B2B市場の実在未証明** | サイジング+10〜15社有償ヒアリングをゲート化。確認まで投資凍結、不発なら比較メディアへピボット。 |
| **個情法27条・名誉毀損** | 監査を「クライアント指定・専用・非再販のスポット受託役務」に再設計。ネガ断定を排し客観指標「検証可能性スコア」へ置換。弁護士DDをフェーズ2着手ゲート化。 |
| **プラットフォーム二重BAN(楽天/Meta ToS)** | 自動投稿を廃し手動コピペ限定、複数ASP・PF多角化、kill-switch実装。 |
| **保有データとB2Bニーズのセグメント不整合／moat不在** | データ拡張PoCを前提化、堀を縦断データ・非公開相関・ワークフロー組込みへ再設計。 |
| **キャッシュ後ろ倒し(B2B着金+45日)** | 最大ドローダウン約−50〜60万を見込み、必要運転資金**約60万を事前確保**。 |

**総括**：下振れは小さいがゼロではない(保守で通期赤字約−38万、運転資金約60万)。誠実さは差別化ではなく前提品質。法的地雷は注記でなく設計で回避する。当たらなければB2Cと比較メディアの小事業に留まる前提での、限定的・段階的な条件付きGO。
