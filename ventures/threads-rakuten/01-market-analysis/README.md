# スレッズ × 楽天アフィリエイター 132名 分析

Threadsで楽天アフィリエイト／楽天ROOMにより「稼いでいる・稼いでいると公言する」個人 **132名**を、Web横断調査で特定・構造化した分析パッケージ。

> ⚠️ 「商品化」の最終方針は内部討議で再設計済み（`../03-business-plan/business-plan.md`／`../02-strategy/direction-debate.md`）。実名の名指し評価の販売・実名リスト販売・初心者向け「稼ぎ方」教材はレッドラインとして棄却。商品化は**匿名化・集計版**に限る。
> ※ファイル名の `affiliators_100` は開発初期の**旧称**で、実データは**132名**。

## 成果物

| ファイル | 内容 |
|---|---|
| **`report.md`** | 分析レポート本体。稼ぎ方の型、収益レンジ、ジャンル分布、マネタイズ導線、注目アカウント（第7章「商品化への示唆」は素案・一部棄却） |
| **`affiliators_100.csv`** | 132名の一覧表（旧称／UTF-8 BOM付き＝Excel/Googleスプレッドシートでそのまま開ける） |
| **`affiliators_100.json`** | 同データのJSON（ダッシュボードのデータソース） |
| **`dashboard.html`** | ビジュアルダッシュボード（検索・絞り込み可能／単体で完結・ブラウザで開くだけ） |
| **`verified-accounts.md`** | Threadsハンドル19名の一次検証メモ（内部限定） |

## サマリー

- 5分類: **教材・ノウハウ販売型 56 / 楽天ROOM型 39 / Threads実践型 25 / コンサル 6 / 検証・批評型 6**
- Threadsハンドル一次確認済み: **19名**
- 誇大の可能性フラグ: **18名**
- 現実的な再現レンジは **月1〜10万円**。月15万超はほぼ教材販売と一体の自己申告。

## 重要な前提

数値の大半は**本人の自己申告**（教材のセールスコピーであることが多い）。Threadsは外部からの検証が難しく、一部は`推定`/`不明`。本パッケージは「額」ではなく**「稼ぎ方の型・導線」を掴む**ための資料。詳細は `report.md` 冒頭の「0. 最初に読む」を参照。
