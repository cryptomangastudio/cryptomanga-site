## 1. 結論：時間キャップつき「実験的GO」

無条件でも通常の条件付きでもなく、**投入時間に月間上限を課した実験としてのみ着手**。現金リスクは小さいが、最大コストである運営者の投入時間（機会費用）を織り込むと保守〜楽観いずれも12ヶ月は赤字。ゆえに「小さく張るオプション」に格下げして運営する。唯一の非対称資産は本業（漫画／IP）由来の一次コンテンツ制作力。

**着手前の最上位ゲート:** 投稿生成ツール（**らくスレ＝https://rakusure.com/ の実プロダクト。旧 rakuaffi はそこへ誘導してクローズ済み**）の楽天／Threads規約適合性を一次確認。グレーならツール系統はスコープ外。

## 2. 収益源（B＋A＋C）と順番

序列は **C → A → B3 → B1 →（規約適合・母数確認後）→ B2**。同時稼働ストリームは最大2。

- **C オウンドメディア（半独立コア／主柱）:** ふるさと納税×食品を旗艦に、比較・ランキング記事。楽天＋複数ASP＋表示広告。真の競合は資本と被リンクで飽和したSEO/比較メディア（価格.com・mybest・ふるなび等）→ ヘッドKWは撤退、ロングテール限定。
- **A 自社実践（収益ゼロ固定）:** ふるさと納税・楽天トラベルを実践し、C記事に実購入・実写真・実数値を供給する一次データ源。金銭リターンは織り込まない。
- **B3 消費者ガイド（高粗利バックエンド）:** 匿名化・集計版の買い切りリファレンス。プライステスト（10本実売）通過が計上前提。
- **B1／B2 LinkDraft（中止候補）:** 無料代替（Rinker等）が強く、本人の楽天口座を巻き込むBAN連鎖の起点にもなりうるため序列最後尾。中止しうる実験扱い。
- **（受け身B2B・付録B）:** 企業案件は**営業しない・財務¥0前提**。B＋A＋Cが育ち自然に問い合わせが来た場合のみ、コンプラ済みのスポット真贋チェック（非再販・客観指標）で受ける＝上振れボーナス。

## 6. 最大リスクと対策

| リスク | 対策 |
|---|---|
| **単一障害点（楽天／Threads規約でB・A・Cが相関倒れ）＋ツールのBAN連鎖** | 「半独立」を直接収益比率でKPI化。楽天依存度を監視し閾値超で投資停止。ツールは口座分離／汎用機能限定／最悪スコープ外。 |
| **機会費用（時間コスト）で経済的赤字** | 月間投入時間に上限、同時稼働≤2。超過月は新ストリーム着手を止める。 |
| **後発SEO不利・AI量産ペナルティ** | ヘッドKW撤退・ロングテール限定。A一次データで肉付けし品質ゲート化。初年度12月ピークは捕捉不可前提。 |
| **無料代替による支払意思の欠如／無料CS赤字** | 固有便益を示せなければB1/B2中止。無料版はノーサポート／自己責任／BAN救済なしを明示。 |
| **法務（ステマ・特商法・定期購入・名誉毀損）** | 全ページに広告開示、住所秘匿、買い切り優先＋解約導線先行実装、小セグメント非公表・弁護士レビュー。 |

**撤退線（数値固定）:** B1無料アクティブ100〜300人未達ならB2非投入。B3は10本実売テスト未通過なら計上せず。本業機会費用の上限侵食時はB2・新ニッチ着手を停止。
