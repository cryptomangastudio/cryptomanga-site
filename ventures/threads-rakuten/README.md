## これは何か

Threadsで楽天アフィリエイトに取り組む「稼ぎ手」132名を分析し、そこから**当社が取るべき事業**を設計する内部プロジェクト。目的は「分析結果を商品化」＝この市場で当社が収益を上げること。

実態を出発点に、事業計画をきれいに立て直した（過去のv2=B2B案／v3=SEOメディア案は `_archive/` に退避）。
