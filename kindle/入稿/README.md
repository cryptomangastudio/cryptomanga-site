# 入稿用docx（KDPアップロード用・自動生成）

本文.md＋入稿用フロント・バック.md から `_build_docx.js`(docx-js)で生成。
構成＝扉→免責(+AI活用の一文)→本文(章=見出し1でKDP目次自動生成)→出典→著者について→奥付(発行=CryptoManga Studios)。

## 再生成
```
cd <scratchpad> && npm install docx && node _build_docx.js
```
原稿を直したら再生成して差し替える。発売日は奥付「［発売日を記載］」を出版後に確定。

## 入稿前の最終目視（KDP側で）
- アップロード→プレビューアで目次(章)・改ページ・扉/奥付を確認
- 表紙は別ファイル(2560×1600・RGB)
