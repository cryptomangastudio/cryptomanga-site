# キーワード需要ワークシート(逆張り「副業やめとけ」)

> 目的: 企画着手前に「その切り口に実需(検索ボリューム)があるか」を数字で確認する。`docs/06`の「次の一手①」に対応。

---

## ⚠️ 実測できなかった理由(正直に)

このワークシートの数値欄が空なのは、**作業環境の制約**による。捏造を避けるため空にしてある:

- **Googleキーワードプランナー本体**は Google広告アカウントへのログインが必須で、エージェント環境からは認証不可。
- **代替ツール(aramakijake.jp / ラッコキーワード / Ubersuggest)や note.com、GoogleサジェストAPI**は、この環境の**組織egressポリシーで403拒否**(到達不可)。回避は禁止されており、報告するのが正しい。
- WebSearchは動くが、検索ボリュームはログイン制ツール内部にあり検索結果スニペットに出ないため拾えない。

→ **実数は、あなたが通常のブラウザで下記手順を実行して埋めてください(15分程度)**。キーワード群と判断基準はこちらで用意済み。

## 実測手順(あなたがブラウザで実行)

**A. Googleキーワードプランナー(推奨・無料)**
1. Google広告アカウントにログイン → ツール → 「キーワードプランナー」→「検索ボリュームと予測のデータを確認する」。
2. 下の「計測キーワード」を貼り付け → 地域=日本、言語=日本語。
3. 各語の「月間平均検索ボリューム」(無料版はレンジ表示: 例 1,000〜1万)を下表に転記。
4. ついでに「新しいキーワードを見つける」に「副業」「FIRE」等を入れ、サジェストされた関連語のボリュームも拾う。

**B. ラッコキーワード + aramakijake.jp(代替・一部無料)**
- ラッコキーワードでサジェスト網羅 → 主要語をaramakijake.jpでボリューム推定。KPが使えない時の代替。

## 判断基準(埋めた後にどう使うか)
- **入口フック**は「需要が中〜大 かつ 大手商用の独占が薄い」語を選ぶ(例: 感情系「疲れた/きつい/やめた」は大手が手薄で、あなたの一次体験が刺さる)。
- **出口テーマ(課金の主力)**は「需要が大きく課金意欲が高い」語(転職・年収)を選び、入口フックから送客する。
- ビッグキーワード(月1万超)単体は競合過多。**「ビッグの感情派生・体験談派生」を取りにいく**のが個人の勝ち筋(`docs/02`)。
- 需要が確認できたら `templates/書籍企画書.md` に落として企画着手。
