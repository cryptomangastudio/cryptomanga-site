# Kindle(KDP)・note 収益化 徹底リサーチ報告書

作成: 2026-07-15 / 手法: 6角度の並列Webリサーチ(日英)→ 上位ソースを抽出しfetch → 各主張を3票の敵対的検証(2/3で棄却)→ 確信度つきで統合。**59エージェント・306ツール実行**。
対象の問い: 「AI生成を正しく開示しつつ、個人がKindleとnoteで **月10万円**、あるいは **年600万円(月50万円)** を稼ぐことの現実性」。

---

## 結論(正直な回答)

**月10万円 = 条件つきで達成可能だが難易度は高い。年600万円(月50万円) = 個人が正攻法・持続的に到達するのは非現実的。**

理由は3つの検証済み事実に集約される:

1. **KUの単価が薄すぎる**: Kindle Unlimitedの既読ページ(KENP)単価は **0.5円を超えることがほぼなく、直近平均で約0.478円/ページ**(※Amazon非公表・著者推定・月次変動、確信度: 中)。150ページ相当の本を1人が最後まで読んでも **約72円**。「大量に薄い本を出してKUで稼ぐ」モデルは、この単価では数理的にほぼ成立しない。
2. **大量出版そのものが制限されている**: KDPは2023年9月以降、標準アカウントで **新規タイトルの登録を1日3点まで**に制限(電子と紙は別カウント、既存本の更新は不算入)。反AI大量出版策で、2026年時点でも継続(確信度: 高)。「一気に100冊」は物理的に不可、日々積む前提になる。
3. **手軽なジャンルほど凍結する**: 2024-2025年、**AI生成の写真集(特に露出度の高い女性写真集)がコンテンツガイドライン違反(わいせつ)でアカウント停止**される事例が多発(確信度: 中)。垢BANは既刊も道連れ=収益ゼロ。「楽して稼げそう」に見える道は、持続性がない。

**したがって現実的な道筋は「大量出版依存」ではない。** AI生成(=開示義務)ではなく **AI"補助"に留めた、専門性・付加価値の高いコンテンツ**を、KDP(薄利・不安定)と note(手数料はあるが単価と継続課金を作りやすい)に **分散**する。**月10万円を現実的な目標、月50万円は例外的成功者の水準**と位置づけるのが誠実な設定。

## 検証済みの事実(2/3以上の敵対的検証を通過)

### ① KDPは1日3点の新規出版キャップを課している 〔確信度: 高 / 3-0〕
標準アカウントは **新規タイトル1日3点**まで(必要に応じ調整あり)。**電子書籍版と紙版は別提出としてカウント**、**既存書籍の更新は算入されない**。影響を受ける出版者はごく少数で、通知・例外申請が可能。2023年9月18日にAI書籍の氾濫対策として発表、撤回の証拠なし。
- 出典: Authors Guild(Amazon KDP公式方針の引用)。Publishers Weekly / Jane Friedman / The Guardian が独立に裏付け。
- **示唆**: 100冊は「日々コツコツ積む」計画にする。1日3点でも月90点は理論上可能だが、品質チェックが追いつかず凍結リスクが上がるだけ。冊数より"当たりの複製"に労力を割く。

### ② AI生成は開示必須、AI補助は開示不要(開示は内部向け・非公開) 〔確信度: 高 / 3-0〕
- **AI生成(AI-generated)**: テキスト・画像・翻訳をAIが作成(実質的な編集後でも)→ **開示必須**。
- **AI補助(AI-assisted)**: 著者自身が作成し、AIをブレスト・編集・校正に使用 → **開示不要**。
- 出版ワークフロー内に「Yes, some of this content is AI-generated」チェックボックスがあり、種別(text/images/translation)を選ぶ。**開示はAmazon内部のみで、購入者には表示されない**。2026年7月時点で有効。
- 出典: Amazon KDP公式コンテンツガイドライン(kdp.amazon.com/help/topic/G200672390)、Authors Guild。
- **示唆**: 「開示すると売れなくなる」は誤解。開示は読者に見えない=正直に申告してノーリスク。さらに **人間の創作的関与を厚くして"AI補助"の水準に持っていければ、そもそも開示義務の外**になり、品質・著作物性・規約遵守を同時に満たす。これが本事業の設計指針(`docs/03`)。

### ③ KENP単価は約0.478円/ページ、0.5円超えはまれ 〔確信度: 中 / 2-1〕
- 「実際は0.5円を超えることはほとんどなく、平均すると0.478円」。複数の日本語月次追跡が直近0.4〜0.5円台を裏付け(2024末〜2025初に約0.48円=過去最高水準との報告)。
- ただし **Amazon非公表の著者推定値**で月次変動。過去は0.6円超、**2025年9月に急落との報告**もあり、将来値は保証されない。
- 「KENP単価は下がっていない(横ばい/微増)」という主張は検証で **棄却(1-2)** → 実態は変動・下振れ含み。
- 出典: wantablog ほか複数の実践者追跡(ブログ級ソース)。
- **示唆**: KUの薄利を前提に数理を組む(`docs/01`)。KUは「読み放題で発見されやすい」利点はあるが、**単価では稼げない**。単品売り(70%ロイヤリティ)との併用・使い分けが要る。

### ④ 手軽なジャンル(AI写真集など)は凍結リスクで持続不能 〔確信度: 中 / 3-0〕
- 2024-2025年、AI生成写真集(特に露出度の高い女性写真集)が **わいせつ・性描写禁止のコンテンツガイドライン違反**で停止される事例が多発。停止メール原文に該当条項が明記。
- 多数の実践者記録(note等)が独立に裏付け。男性/露出なし写真集の停止事例は単一ソースで裏付け弱いが、Amazonの執行の一貫性のなさを踏まえ否定材料もない。
- 「Amazonは基準でなく時期で一斉取締り」という説は検証で **棄却(0-3)**。
- 出典: hibi-create ほか、Amazon KDP公式コンテンツガイドライン(kdp.amazon.co.jp)。
- **示唆**: "楽して稼げる"と喧伝されるジャンル(AI写真集・アダルト隣接)は避ける。**垢BANは1冊で全滅**。規約遵守(`docs/03`)は税務以前の生命線。

## 十分な裏付けが取れなかった論点(=断定しない)

以下は今回の敵対的検証で「確定クレーム」として提示できなかった。一般に流布する知識やKDP/note公式の建て付けとして `docs/01`〜`05` に反映しているが、**一次ソースで各自確認すべき未確定事項**として明示する:

- **KDP印税の実額分布**(35%/70%の適用条件、KDP Selectの詳細、日本の個人著者の収益中央値・上位分位)
- **「100冊で月10万円」の完全な数理的裏付け**(1冊あたり平均月収の実測)
- **note の手数料構造の実率**(プラットフォーム手数料+決済手数料)と、実際に売れるコンテンツ類型
- **日本の著作権法上のAI生成物の扱い**(人間の創作的寄与の程度による帰属・保護範囲)
- **副業としての確定申告・税務**(20万円ルール、雑所得/事業所得区分、経費計上)

→ `docs/01_収益シミュレーション.md` の数値は、確定した0.478円/KENPと一般的なKDP印税建て付けに基づく **試算**であり、実測の収益分布データではない点に注意。

## この報告が土台に与える設計指針(まとめ)

1. **量産で薄く稼ぐ、は捨てる。** KENP 0.478円・1日3点・凍結リスクが、それを数理的にも運用的にも否定した。
2. **勝ち筋は「AI補助 × 専門性・付加価値 × KDPとnoteの分散」。** 開示義務の外に出る品質を作り、薄利のKUに依存しない。
3. **月10万を現実目標、年600万は例外水準。** 目標を段階化し、まず1冊を"売れる"にしてから型を複製(`docs/02`)。
4. **規約遵守は最優先。** 垢BAN=全滅。手軽なジャンルの誘惑に乗らない(`docs/03`)。
