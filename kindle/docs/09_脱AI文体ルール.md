# 09 脱AI文体ルール —「AIっぽさ」を消して、読まれる文章にする

> 全原稿(Kindle・note)に適用する必須ルール。CryptoManga Studiosの制作にある「**脱AI文体の番人**」を、この事業でも正式な査読視点として運用する。
> 発端: 第1巻まえがき初稿への代表の批評(2026-07)。「稼ぎ/損失の数字・家族の教訓・お節介アドバイスをミキサーにかけた、無難で綺麗なだけの副業本まえがき」。**書いた本人の顔と体温が見えない**ので読む気が起きない、という指摘。全部その通り。

---

## AIっぽさを生む5つの病(=禁止事項)

### 1. テンプレの「光→影→共感」構成
- ✗ 「実績で引く→挫折を語る→共感で締める」の黄金パターン。「稼いだ(光)/溶かした(影)」を綺麗な落差で並べる。
- ◎ **対比を綺麗に作らない**。語り順をわざとズラす(損失から始める/時系列を崩す/オチを先に言う)。教科書構成を意図的に壊す。

### 2. 記号的なエピソード(体温ゼロ)
- ✗ 大事件を"数字"として処理する。「700万溶かした」で終わり。感情語(絶望した/つらかった)だけで済ます。
- ◎ **その時、体と生活がどうだったか**を必ず入れる。胃、手、眠れたか、その日の家、誰に言えたか。**抽象の感情語でなく、身体・行動・具体物**で。ここは本人の実記憶が必須(AIが捏造しない)。

### 3. シグナルフレーズ(「これから本音ですよ」の合図)
- ✗ 「同じ口で、正直に言います」「ここだけの話」「正直に言うと」「実は」。本音に入る前の"合図"。
- ◎ **合図せず、いきなり言う**。本当のことは、前振りなしで置く。

### 4. 読者のコントロール(お節介・先回り)
- ✗ 「ちょっとだけ待ってください」「どうぞ肩の力を抜いて」「〜と思いませんか?」「気づきませんか」。読者の感情・態度を指定/代弁/同意誘導する。
- ◎ **読者の感情を指定しない**。事実と自分の話を置いて、感じ方は読者に委ねる。呼び止め・気遣いの押し付けをしない。

### 5. 整いすぎ(ノイズがない)
- ✗ 全文が均等な心地よいリズム。文頭「——」でタメ(多用)。「A。B、C、Dも。」の3〜4拍子の綺麗な羅列。引っかかりがなさすぎる。
- ◎ **ノイズを残す**。一文の長さを不均一に。ぶっきらぼうな断片、言い切り、主語の省略、小さな脱線。整いすぎたら、わざと崩す。「——」の連発禁止(1章に1回あるかないか)。3拍子の羅列を避ける。

## ペルソナ(語り手)を一つに固定する
- ✗ 「溶かしました」(くだけ)の直後に「肩の力を抜いて」(スマートなビジネス書作家)。口調が混ざると"不気味の谷"になる。
- ◎ **著者の実際の喋り方に寄せる**。この著者は、カジュアル・直球・自虐まじり(音声取材の口調)。ビジネス書のすまし顔にしない。全編、同じ人が喋っている状態を保つ。

## 避けたい"AI定番語"(乱発しない)
「〜という選択」「〜のポテンシャル」「本質」「〜と地続き」「〜に他ならない」「まさに」「ある種の」「〜と言えるでしょう」。滑らかで無難な言い回しは、AI臭の温床。

## 編集時の「脱AI文体の番人」パス(全原稿で実行)
`docs/07`§4の検証プロトコルに、この査読視点を1枠として加える。原稿を読み返し、次を潰す:

- [ ] 光→影→共感のテンプレになっていないか(なっていたら順序を崩す)
- [ ] 大事件に"体温"(身体・行動・具体物)があるか。なければ本人の実記憶を入れる(捏造しない)
- [ ] シグナルフレーズ(「正直に言うと」等)がないか
- [ ] 読者コントロール(「待って」「肩の力を抜いて」「〜ですよね?」)がないか
- [ ] 「——」の多用・3拍子の羅列・整いすぎがないか(ノイズを足す)
- [ ] 語り手の口調が最初から最後まで一人か
- [ ] AI定番語を乱発していないか

> 最重要: **本人の実記憶(体温)なしに、AIだけで"生々しさ"は作れない**。生々しさが要る箇所は、必ず取材(`取材質問リスト`)で本人の本当のことを引き出してから書く。想像で埋めない。

## 追加ルール(第1巻リライトで判明した4つのバグ・2026-07)

代表が第1巻を全面リライトして特定した、まだ残っていたAI臭。この4つは特に強く効く:

### バグ1: 一人称の「セルフ実況(ひとりノリツッコミ)」禁止
- ✗ 「……自己紹介がいきなり物騒ですね。落ち着いて〜」のように、自分をカメラで引いて実況・ツッコミする。YouTube台本のノリ。この一言で真剣味が消え、「キャラを演じてるだけ」と見破られる。
- ◎ 内省・告白は、自分を茶化さない。照れ隠しのツッコミを入れない。(※在庫の「すみません」的な"謙虚な詫び"はOK。違いは「自分を実況・演出しているか」)

### バグ2: エピソードは「概念(記号)」でなく「五感の泥臭いディテール」で
- ✗ 「ピクミンを一緒にやりたかった。笑いたかった。それだけ」=具体名を"家族団欒の記号"として置いただけ。
- ◎ 本人にしか見えない具体を入れる。例:「Switchのコントローラーを持って寄ってきた子を『今忙しいから』とあしらい、その子がコントローラーを片付けた**プラスチックの小さな音**」。この"ノイズ"が体温になる。→ 取材で実際の場面を引き出す。

### バグ3: ペルソナを割らない=「お節介なカウンセリング」禁止
- ✗ 「700万溶かして妻に言えず怯えてた情けない男」が、章が変わると急に「罪悪感は捨てていい」「あなたは悪くない、構造の問題」と、救済者・カウンセラーに変貌する。読者は「お前どの立場で言ってる?」と不一致(不気味さ)を感じる。AIの「ユーザーを肯定してあげる」設定がキャラを壊す。
- ◎ **最後まで「情けない反省者」の視点**を保つ。読者を上から救わない。同じことを言うにも「僕もそうだった/僕はこう思う」と、自分の失敗の地平から語る。読者をハグしにいかない。

### バグ4: スマホ縦読みを意識しすぎた「短文・平板リズム」をやめる
- ✗ 「線を割った。終わり。」「飯が食えなかった。夜も眠れなかった。頭の中は後悔だけ。」=一文を極端に切り、「〜でした」「〜です」を短スパンで反復。インスタのカルーセル/スライド広告の文字を読まされてる感。幼児退行した薄さ。
- ◎ 大人の文章の**タメとリズム**を保つ。短文は"ここぞ"の一撃だけ。基本は、一文にある程度の情報と呼吸を持たせて、読ませる。

> この4つは、`docs/07`§4の「脱AI文体の番人」パスに全部含める。

## 追加ルール5: 比喩の「劇画調クリシェ」を禁止(第1巻リライトで判明)

熱量を上げようとすると、AIは手垢のついた比喩(クリシェ)を引っ張ってくる。これが最大の"嘘くささ"を生む。

### 禁止する比喩の型
- ✗ **スケールが生活から浮いた比喩**: 「電子の海の藻屑」「大嵐の登山」「命綱なしの登山」「奈落の底に叩き落とされる」「地獄の扉」「猛獣」。38歳の会社員の日常(小売・育児・スマホ)を語っているのに、急に大自然/アドベンチャーのスケールに飛ぶと、話全体が嘘っぽくなる。
- ✗ **劇画調・中二病・情報商材LP臭**: 「脳汁ドバドバ」「狂気と欲望が渦巻く」「魂を吸い取られる」「屍が転がる」「反吐が出る」。本当に絶望した人間は、もっと静かで、乾いた、冴えない言葉で惨めさを語る。
- ✗ **芝居がかった呼びかけ**: 「ねえ、あなたに聞きたい」「あなたが今、必死になって〜」。自己啓発セミナーの講師がマイクで語る人工的な響き。

### 正しい比喩は「半径1メートルの生活」から採る
段ボール、レジ、おむつ、スマホのバッテリー残量、エクセル、更地、アプリのインストール——読者の日常の道具で例える。
- 例: 「乾いた雑巾のような毎日」→「**充電が5%しかないスマホで、さらに重いアプリを立ち上げるようなもの**」
- 例: 「記事が奈落の底へ叩き落とされた」→「**朝起きたら、自分の店が跡形もなく取り壊されて、更地になっていた**、みたいな感覚」
- 例: 「700万が電子の海の藻屑に」→「**スマホの画面の上で、ただの『0』というフォントになった**」

### 呼びかけは芝居にしない
「ねえ、あなたに」でなく、「少しだけ胸に手を当てて考えてみてほしい」「〜していませんか」程度の、生身のトーンで。

## 追加ルール6: 「行儀の良さ」を崩す(第1巻v5で判明)

ディテールとクリシェ除去をやっても、まだAI臭が残ることがある。原因は「構成のつなぎと比喩の使いどころが、教科書のように等間隔で綺麗に収まりすぎている」=**泥をかぶったのにスーツを着たままのお行儀の良さ**。

- ✗ **綺麗すぎる接続と構成**: 「誤解しないでほしいのですが」「確かに〜。でも〜。だからこそ〜」の"読者への気遣いセンサー"。→ ◎ 感情の起伏をわざと不格好に。気遣いの前置きを削り、いきなり言う。
- ✗ **データを"講義"にする**: 「◯◯社の2025年の調査によると〜」が本文に挿さると、ニュース原稿が紛れ込んだように浮く。→ ◎ **データは著者の悔しい主観に溶かす**(「要するに、毎晩必死でキーボードを叩いても、9割はお小遣いにもならないという冷たい現実」)。**出典は本文でなく巻末の『出典・参考データ』にまとめる**(信頼性は担保しつつ本文は体温を保つ / `docs/07`)。
- ✗ **箇条書き(スライド資料感)**: `*` のリストは"プレゼン"の匂い。→ ◎ 文章のリズムの中で読ませる(「子どもと過ごす時間であり、パートナーと話す時間であり、睡眠や健康です」)。
- ✗ **クリシェの残党**: 「無風の墓場」「屍」「地主」「猛毒」等ビジネス書語。→ ◎ 現場・生活の比喩へ(「バックヤードの隅で埃をかぶった売れ残りの段ボール」「大家さんに鍵を換えられて締め出されたアパート」)。

> データの扱いは重要な両立: **本文=主観に溶かす / 巻末=出典を明記**。数字を裏付けなしで断定しない(`docs/07`)。
