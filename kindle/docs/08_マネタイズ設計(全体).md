# 08 マネタイズ設計(全体)—「信頼を売って、地続きのものだけで回収する」

> このブランド「消耗しない所得の増やし方」の収益の全体像。**最大の資産は"逆張り誠実の信頼"**。それと矛盾する売り方(高額情報商材・過剰アフィリ)は、資産そのものを壊すのでやらない。
> 〔2026-07 確定〕以下は代表判断で確定した方針。

---

## 大原則
**信頼を売って、信頼と地続きのものだけで回収する。** 「副業やめとけ(=煽らない)」と名乗る以上、煽り側の手法(高額商材・アオリ・過剰アフィリ)を使った瞬間にポジショニングが死ぬ。誠実さ=売上、なので誠実さを削らない。

---

## 1. チャネルの役割分担 〔確定〕

**同じ原稿を両方に丸ごと二重掲載しない。1つの制作物を分解して二度使う。**

| チャネル | 役割 | 中身 | 課金 |
|---|---|---|---|
| 無料note ＋ X | 集客・人格を立てる | 回0など。試し読み・共感 | 無料 |
| **Kindle(6冊+複製)** | **資産(主役)** | 検索/KUで見つかる"本" | ¥500前後・KU |
| **note有料** | 濃い付録＋継続課金 | Kindleに載せない実額・失敗の詳細・最新・個人的深掘り | 単発**¥980**（2026-07改定。note公式分析の売れ筋=読み物系¥980に合わせる。¥300台は安売りシグナルのため使わない） |
| **note メンバーシップ/月額マガジン** | ストック収入(最重要) | 継続の深掘り・限定・伴走 | 月額 |

**分解の例**: Kindle本の1章を無料note化(試し読み→本へ)。本に載せきれない濃い部分・実データ・生々しい失敗をnote有料へ。

## 2. 収益の柱(優先順)〔確定〕
1. **note継続課金(メンバーシップ/月額マガジン)** — 一番安定・ファン化・ストック
2. **Kindle 6冊+複製** — 検索流入の資産(薄利だが積む)
3. **note単発有料** — Kindleの濃い付録
4. **転職アフィリ(出口一致のみ・控えめ・開示)** — 下記3
5. **LINE/メルマガ** — 導線のみ(下記4)

## 3. アフィリエイト方針 〔確定: 出口一致のみ・控えめ〕
- **原則、主張と矛盾するアフィリはやらない**(「副業やめとけ」と言いながら副業商材、は自滅)。
- **例外的にOK**: 出口テーマ「転職」と一致するもの=**転職・退職代行**サービス。著者のブログ実績(高単価アフィリ=転職・退職代行)とも合致し、主張と地続き。
- **やり方**: 「副業より転職」の文脈で**控えめに・PR表記で開示して**紹介。リンク詰め込みや煽りはしない。
- **Kindle本文**: KDP規約上、外部リンク・宣伝は慎重に(過剰はNG)。アフィリは主にnote/ブログ側で。
- **採用サービス〔2026-07 確定・著者が実際に利用/今もスカウトが届く〕**: 総合型=**リクルートエージェント**(求人数・相場/市場価値の確認)／ハイクラス・スカウト型=**ビズリーチ**(登録して声を待つ)。第2巻の実体験(第8章)と一致＝出口一致で自然。
  - **置き場所の線引き〔2026-07-18確定・著者判断=受け皿はnote〕**: **Kindle本文＝名前のみ(リンクなし)** ／ **ASPリンクはnoteの無料記事に設置**(独自ブログは作らない。ASP側はnote受け入れあり=afbは公式note向けガイドを公開・A8はサイト無し登録可・もしもはnote実績あり) ／ **LINEはnote記事URLを配るだけ**(アフィリ直リンクの配信はBANリスクのため絶対にしない)。
  - **note設置の運用ルール（note側規約がグレーなためリスクを下げる4条件）**: ①記事冒頭(ファーストビュー)に「PR/広告を含みます」表記 ②**有料記事の中にはASPリンクを入れない**(無料記事のみ) ③体験談ベースの記事にする(宣伝だけの記事にしない=コンテンツポリシー対策) ④画像は自前のみ(無断転載はASP登録削除の実例あり)。万一noteで問題が出たら、その時点で無料ブログへ退避(リンク差し替えはLINE/note導線側で吸収可)。
  - 実リンク=著者がASP登録(**afb=note公式ガイドあり・ビズリーチ取扱い→本命**／A8=即日・現況実測用／もしも=予備)。本命案件は**ビズリーチ**(成果地点=無料登録)。必ずPR開示・煽らない・登録無料/強制でない旨を添える。広告主都合の打ち切りに備え複数ASP×複数広告主でヘッジ。詳細=`docs/research/2026-07_転職アフィリ取得手順(調査)`。

## 4. LINE/メルマガ・情報商材 方針 〔確定: 導線のみ・商材は売らない〕
- **公式LINE/メルマガはOK**。用途は「新刊案内・無料の価値提供・再訪導線・(将来)少額コミュニティの案内」まで。
- **高額情報商材は売らない**。「副業やめとけ」と言いながら情報商材を売るのは自己矛盾で、"煽る側"と同一視された瞬間に信頼(=資産)を失う。第1章で「僕は商材屋じゃない」と名乗る手前、ここは絶対に踏まない。
- **CTAの書き方 〔2026-07 確定〕**: 「売り込みはしません」と**防衛線を張らない**。誠実に見えて、実は「読者コントロールの裏返し」で、かえって「本当に?」と身構えさせる(`docs/09`ルール4)。→ **フラットに価値提供**する(「本編に載せきれなかった、そのまま使える具体をまとめた。同じ遠回りをしたくない人はどうぞ」)。潔い読後感のほうが、この逆張り誠実ブランドに合う。

## 5. 導線(ファネル)まとめ
```
無料note/X(集客・人格)
   → Kindle(本=資産。安く広く読ませ信頼を作る)
   → note有料/月額(濃い付録・継続課金=主な回収)
   → (出口の文脈で)転職アフィリを控えめに・開示して
   → LINE/メルマガで再訪・新刊案内(商材は売らない)
```

## 6. 収益規模の前提(再掲・正直に)
- **月10万=現実目標、月50万=非保証**(`docs/01`/`06`)。煽り系の成功譚は検証で全棄却。
- 薄いKDP単価(KENP≒0.478円)に依存せず、**note月額＋Kindle資産＋(控えめな)転職アフィリ**の合算で組む。
- 当たった型を複製して積む(`docs/02`)。

---

> この方針から外れる誘惑(手っ取り早い高額商材・広範なアフィリ・煽りコピー)が出てきたら、**"信頼という資産を削っていないか"** で判断する。削るならやらない。
