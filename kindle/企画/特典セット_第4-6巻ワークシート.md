# ① 【第4巻特典】夫婦の「時間の家計簿」＆家事の棚卸しシート

> 『「家族のため」が、妻を追い詰めていた。』の実践ツール。副業を足す前に、まず"時間の偏り"を二人で見える化する。

## A. 時間の家計簿（一日24時間の引き算）
夫・妻それぞれ、平日の1日を書き出す。**犯人探しではなく、二人で同じ表を眺めるのが目的**。

| 項目 | 夫（時間） | 妻（時間） |
|---|---|---|
| 睡眠 | 　 | 　 |
| 仕事（通勤含む） | 　 | 　 |
| 食事・入浴・身支度 | 　 | 　 |
| 家事（名もなき家事も！） | 　 | 　 |
| 育児・子の世話 | 　 | 　 |
| **自由に使える時間（残り）** | 　 | 　 |

→ 「自由に使える時間」の欄が、**二人ともほぼゼロ**なら、そこに副業を足すのは無理。まず下のBで、家事を分け直す。

## B. 家事の棚卸し（"名もなき家事"を書き出す）
「やって当たり前」で見えていない家事を、あえて全部書き出す。担当に◯を。

- □ トイレットペーパー等の在庫管理・買い足し（担当：夫 / 妻 / 共有）
- □ 子の服・持ち物のサイズ替え・準備（担当：　）

- □ 献立を考える・食材の段取り（担当：　）
- □ 体調の異変にいち早く気づく（担当：　）
- □ ゴミの分別・出す曜日の管理（担当：　）
- □ （書き足す：　　　　　　　　　　）

→ ◯が**片方に偏っていたら、それが"感謝されない疲れ"の正体**。「手伝う」ではなく「二人の共通の仕事」として、1つでも移す。

## C. 今週の一歩
- □ 家事を1つ、相手から自分へ移す

# ② 【第5巻特典】「これで十分」ラインの決め方＆心の余白チェック

> 『稼いでも稼いでも、なぜ楽にならないのか。』の実践ツール。"もっと"の追いかけっこから降りるために、ゴールを他人の手から自分の手に取り戻す。

## A. 「十分ライン」を、自分の言葉で決める
他人（SNS）に決めさせない。自分の生活で、具体的に書く。

- 我が家が、普通に暮らして、たまに旅行に行けて、子どもにやりたいことをやらせてやれる——
  そのために必要な、**月の手取り**は：**　　　　万円**
- そのラインを超えたお金で、本当にやりたいことは：**　　　　　　　　**
- 逆に、これ以上の「もっと」のために、**差し出したくないもの**は：**　　　　　　　　**（例：睡眠・家族の時間・健康）

→ この3つを書けたら、SNSで「月100万」を見ても、「あの人はすごい。でも僕の十分はここ」と言える。

## B. 心の余白（メンパ）チェック
タイパ（時間効率）だけでなく、**メンパ（心の余白）**が削れていないか。当てはまるものに☑。

- □ 予定の入っていない時間が、なんだか怖い
- □ 動画は倍速、移動中も何か"生産的なこと"で埋めないと落ち着かない
- □ 最近、ささいなことで家族にイラッとした
- □ 「稼がなきゃ」で、夜、寝つきが悪い

→ 2つ以上☑なら、メンパが赤信号。下のCを1つ。

## C. 降りるための地味な一歩
- □ 体を動かして心を止める（夜の散歩・ランニング・サウナなど、スマホを置く時間を1回）
- □ 途中で止まっている"未完成"を、1つだけ、60点でいいから終わらせる
- □ 何もしない時間を、罪悪感なく過ごす（子どもと、意味もなくだらだら）

# ③ 【第6巻特典】副業の「本当の時給」計算シート（隠れコスト付き）

> 『その副業、時給にすると300円かもしれない。』の実践ツール。始める前に、電卓で1回。

## A. 表の時給を出す（割り算ひとつ）
- その副業で、ひと月に得た（得られそうな）お金：**　　　　円**（Ａ）
- そのために使った（使う）時間の合計：**　　　　時間**
  → ★「手を動かした時間」だけでなく、**調べた時間・悩んだ時間・SNSを気にした時間も全部**足す
- **表の時給 ＝ Ａ ÷ 時間 ＝ 　　　　円／時**

## B. そこから引く（隠れコスト）
- □ 税金：所得が増えれば所得税・住民税が引かれる（Ａはまるごと残らない）
  ※20万円ルールは所得税の話。住民税は20万円以下でも申告が必要。最新は税務署／税理士へ。
- □ 事務の手間：帳簿・確定申告の時間も、本当は"使った時間"に足す
- □ 本業への悪影響：寝不足で本業のパフォーマンスが落ちていないか

## C. 数字に出ない、いちばん大きいコスト（意識に入れる）
金額にはできない。でも、時給の裏で差し出しているもの：
- □ 気力（本業のあとの、残り少ないバッテリー）
- □ 家族の時間（子が「遊ぼう」と来る時期は、驚くほど短い）
- □ 健康・睡眠

→ 「表の時給 ◯◯円」の裏で、**これらを差し出して、なお割に合うか？**

## D. 判定
- その副業の目的は？ → □ お金のためだけ（→たいてい割に合わない。第2・3巻の"本業に賭け直す"が近道）
  　　　　　　　　　 □ スキルへの投資 or □ 純粋な楽しみ（→時給が安くても、やる意味がある）
- □ 「やらない」も、立派な選択。ただし**若くて失うものが無いなら、時給を気にせず今すぐやれ**（第1巻）
