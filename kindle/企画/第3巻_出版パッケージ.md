## 2. 商品説明（KDPにそのまま貼る）

```
「黙って真面目にやっていれば、いつか評価される」――僕はそれを信じて、15年、据え置きでした。
評価は、あなたが思うほど「成果」では決まっていません。「見え方」で決まる部分が、ずっと大きい。
本書は、昇給の“負け組”だった著者が、静かに上がっていく人たちを横目で観察して盗んだ、地味な習慣の記録です。

────────────────

賃上げの時代とはいえ、その多くは「全員一律の底上げ」。そこからさらに、個別に評価されて上乗せされる人と、されない人がいます。
同じ職場で、同じような仕事をしているのに、なぜか静かに上がっていく人は、何が違うのか。
本書は、転職（第2巻）とは別の道――「今の会社で、家族の時間を削らずに手取りを上げる」ための、見せ方と習慣の話です。

【この本に書いてあること】
・評価は「成果」でなく「見え方」で決まる、という身も蓋もない現実
・昇給交渉の準備・タイミング・言い方（脅しでなく相談で）
・上司を敵でなく“味方（あなたの代理人）”にする
・副業に注ぐはずの気力を、本業の一点に集中させる割の良さ
・静かに評価される人の地味な習慣（記録・半歩先・機嫌の一定）
・「頑張りました」をやめ、数字で自分を語る技術

【こんな人に読んでほしい】
・今の会社はそこまで嫌いじゃない。年収だけ、もう少し上げたい人
・真面目にやっているのに、なぜか評価されないと感じている人
・転職（第2巻）の前に、まず今の場所で打てる手を知りたい人

【正直に、向いていない人】
・「その業界・会社の天井」がすでに低い人
　（その場合の答えは、本書ではなく第2巻＝転職です。本書内でもそう明言しています）

著者は元・小売の販売員（15年）。その場所では上げられず、最後は転職で年収を400万→700万に。だからこそ、「上がらない側」の景色から、正直に書いています。

※本書は、著者の実体験を軸に、公的機関や大手調査会社の統計データ（出典明記）を交えて構成しています。
```
