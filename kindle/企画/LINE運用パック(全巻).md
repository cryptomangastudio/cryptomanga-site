## 2. ステップ配信（手動ブロードキャストの文面ストック）

> LINE無料枠では「あいさつ（自動）＋手動ブロードキャスト」で回せます。以下を、追加が増えてきたタイミングで順に送る（目安の間隔つき）。

### 2-1. 初回（追加から3〜4日後）
```
先日は、追加ありがとうございました。特典のPDF、目を通してもらえたでしょうか。

一つだけ、本でいちばん伝えたかったことを、改めて。

「お金を増やすこと」より、「消耗しないこと」を先に置く。
たったこれだけで、選ぶ手段の順番が変わります。副業をもう一個足す前に、まず"いまの本業の価値を上げられないか"を考える。遠回りに見えて、これがいちばん速かったんです。

今日も、お疲れさまでした。
```

### 2-2. 価値提供（追加から10日前後）
```
今日は、僕がいちばん高い授業料を払って学んだことを、一つだけ。

「稼ぎだけを数えて、払ったコスト（税金・時間・家族）を見ない」。
この計算のクセが、僕を6年も沼に沈めました。1000万稼いだはずが、正直に足せば手元はむしろマイナス。しかも、いちばん近い家族の時間まで削っていた。

もし今、副業を足すか迷っているなら。始める前に一度だけ、その"本当の時給"を電卓で出してみてください。それだけで、見え方が変わるはずです。
（詳しくは⑤『時給300円』に書きました）
```

### 2-3. note（受け皿）へ（追加から2〜3週間・アフィリ導線）
> ⚠️ 送るのは**プロフィールURLでなく、アフィリ記事そのもののURL**（#5『副業より転職だった』公開後に、そのnote記事のURLへ差し替え）。それまでは他の集客記事（#1/#3/#4/#6/#8/#9のいずれか）のURLで代用可。
```
本編には載せなかった「その後」の話を、noteに書きました。
僕が実際に登録して、相場を確かめた転職サービスのことも、正直に置いてあります（PR表記あり）。

▼ ［note記事URL：公開後に差し込み（プロフィールでなく記事URL）https://note.com/kidokyari 配下］

売りつけるものは、何もありません。同じ遠回りをしてほしくないだけの、記録です。
```

## 4. （任意）特典を出し分けたい場合

> 「①〜③の人には特典1、④〜⑥の人には特典2」を分けたいとき。あいさつでは両方渡さず、キーワードで出し分ける。

あいさつメッセージ末尾にこう足す：
```
どちらの本から来てくれましたか？
・転職／昇給の話（①〜③）→「転職」と送ってください
・家計／時給／もっと の話（④〜⑥）→「家計」と送ってください
```
→ キーワード「転職」に特典1リンク、「家計」に特典2リンクを自動応答（6.参照）。
※まずは **1.の"両方渡す"でシンプルに始める**のを推奨。出し分けは友だちが増えてからで十分。

## 7. 運用ルール（守る）
- 配信は**低頻度**（月数回まで）。売り込みを混ぜない＝信頼が資産（`docs/08`）。
- **アフィリ直リンクをLINEで配らない**。必ずnote無料記事のURL経由（BAN・規約リスク回避）。
- 情報商材・高額サロンは売らない（方針＝導線のみ）。
- 声は本文と地続き（`docs/09`）。煽らない・脅さない。
