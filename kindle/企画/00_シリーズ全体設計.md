# シリーズ全体設計 —「副業やめとけ」逆張り事業のラインナップ

> 目的: 1冊ずつ書き始める前に、**シリーズ全体を上から設計**する。何冊・どのタイトル・どう噛み合い・どの順で書くか。
> 根拠: `docs/06`(ポジショニング)/`tracker/キーワード需要ワークシート`(実測・競合空白)/`docs/01`(数理)。
> 方針の芯: **量産で薄く稼ぐは捨てる。尖ったコアを数冊作り、"当たる型"を複製して積む**(`docs/02`)。

## 0. ブランド(シリーズ)コンセプト 〔確定〕

> **シリーズ名(正式): 「消耗しない所得の増やし方」**
> キャッチコピー: **「稼ぐより、消耗しない。」**
> ——副業で1000万稼いだ僕が、いま"やめて本業と家族に賭け直す"までの記録と、その方法。

**確定事項(2026-07)**: シリーズ名=消耗しない所得の増やし方 / 第1弾=6冊(コア3+派生3) / タイトルの尖り=**告白型のまま**(強すぎず・炎上しにくく・人格が立つ)。

### 背骨の主張(★全6冊が共有する芯・著者の本音に基づく)
**「副業やめとけ」の全否定ではない。** 著者の本音はもっとニュアンスがある(→ `著者プロフィール(一次情報).md`):

> **敵は「副業」ではなく、「今のライフステージに合わない副業の"上乗せ"」。副業には"やるべき時期"がある。**

- **若い人・独身・結婚前 → 今すぐガンガンやれ**。若いほどスキル・人脈・お金・"応援されやすさ"が手に入る。副業は楽しく、価値観も広がる(著者自身がそう得てきた)。
- **30代・子持ちで"新たに上乗せ" → 正直もう遅い**。多忙な本業+家事育児に重ねるのはコスパが悪い。やるなら無理のない範囲で。
- **所得を本気で増やしたいなら、本業(転職・昇給)に賭け直す**方が合理的。

この芯があることで:
1. **「稼いだお前が言うな」に完全反論**できる(「若い時にやったから今がある。でも"今のあなたの時期"には勧めない」=矛盾でなく時期論)。
2. **全否定より刺さり、炎上しにくい**(副業肯定派も敵にしない。読者を選別=ファン化)。
3. **6冊すべてに一貫した世界観**が通る(若者向けの巻/子育て世代向けの巻、と対象を分けても芯は同じ)。

> ⚠️ 執筆時の注意: 単純な「副業=悪」の煽りにしない。**「時期とステージ」で語る**。タイトルは告白型フックのまま、中身はこのニュアンスで深める。

- 一貫する人格: **稼いだ当事者だからこそ「やめとけ」と言える**(空白a・実質競合なし)。
- 一貫する主張: 所得を増やす順番は「副業を上乗せ」ではなく「**消耗を減らして、本業と時間の質を上げる**」。
- 一貫する感情: 自慢でなく**後悔**。攻撃でなく**告白**。

## 1. 何冊書くか(正直な結論)

**「100冊」は目標にしない。** 検証(`docs/research`)で、薄い量産は数理的にも規約的にも沈むと確定済み。正しいのは:

```
第1弾ラインナップ = 尖ったコア3部作 + 型を複製した派生3冊 = 計6冊
   ↓ この6冊で「当たる型」を市場に確かめる
第2弾以降 = 当たった型を "対象×職種×年代" で複製して積む(下記マトリクス)
   ↓ 冊数はここで初めて伸びる(当たりの複製としての冊数)
```

- **まず6冊**。これは「棚として成立する最小のシリーズ」であり、入口→出口のファネルを1周させられる単位。
- 100冊は"目標"ではなく、**当たった型を複製した"結果"**として後から到達しうる数(そのための複製ルールは §5)。

## 2. ファネル設計(6冊がどう噛み合うか)


## 3. ラインナップ(タイトル先行・全6冊)

各巻: 分量2〜3万字 / 章はスマホ縦読み前提 / 尖り=誇大でなく「稼いだ当事者の告白」で刺す。

### ● コア3部作

#### 第1巻(入口アンカー)★最初に書く
- **タイトル: 「副業、やめました。——1000万稼いで700万溶かした僕が、あなたに勧めない理由」**
- フック: 「稼げなかった人」でも「商材屋」でもない、**稼いで溶かした当事者の"やめとけ"**。稼得だけでなく"700万溶かした"損失を出すことで一段強い掴みに。読者へは「家族」でなく「あなた」と直接向ける。
- 埋める空白: a(稼げたのに家庭を失った当事者告白)/ 検索: 副業やめとけ・やめた・疲れた
- 分量: 約2.5万字 / KU入れる・¥500
- 章スケッチ:
  1. 「副業やめとけ」を、稼いだ僕が言う
  2. 僕が触った副業と、稼いだ金額の全記録【一次情報】
  3. 一番高くついたもの——家族の時間の"時給"
  4. データが示す「割に合わない」の正体【要確認:統計】
  5. 副業が向く人・向かない人(独身/子育て/年代)
  6. 「始められない罪悪感」を捨てていい理由
  7. じゃあ所得はどう増やす?(→第2・3巻へ橋)
  8. おわりに:消耗しない、という選択

#### 第2巻(出口・本命/転職)
- **タイトル: 「副業より、転職でした。——数百時間を溶かした僕が気づいた、年収の上げ方」**
- フック: 副業に注いだ数百時間 vs 転職1回を**時給で並べる**衝撃。「溶かした」で第1巻と地続き。
- 埋める空白: c(入口→出口を同一書き手で一本道)/ 検索: 転職 年収アップ(~1,000・本命)
- 分量: 約3万字 / 単品¥700〜(実用価値=課金理由)
- 章スケッチ:
  1. 副業100時間 vs 転職100時間:時給で比べる
  2. 年収は"一度上げれば毎月続く"という複利
  3. 稼いだ経験は転職市場でこう効いた【一次情報】
  4. 何歳でも遅い?転職の現実【要確認:データ】
  5. 家庭を壊さない転職活動の進め方
  6. やってはいけない転職(消耗の再生産)
  7. エージェントの使い方・年収交渉
  8. 実例:僕の年収の変化【一次情報/要確認】
  9. おわりに

#### 第3巻(出口・本業/昇給)
- **タイトル: 「静かに、昇給する。——転職せずに年収を上げる人の、地味な習慣」**
- フック: 転職しなくても上がる。「評価は成果でなく"見え方"」という身も蓋もない現実。
- 埋める空白: 「本業に集中」系(競合薄)/ 検索: 昇給 交渉・年収を上げる方法
- 分量: 約2.5万字 / KU or 単品¥500
- 章スケッチ:
  1. 転職だけが答えじゃない
  2. 評価は"成果"でなく"見え方"で決まる現実
  3. 昇給交渉の準備・タイミング・言い方
  4. 上司を敵でなく味方にする
  5. 副業の気力を本業の一点集中へ
  6. 静かに評価される人の習慣
  7. 数字で自分を語る技術
  8. おわりに

### ● 派生3冊(型の複製・第1弾)

#### 第4巻(共働き)
- **タイトル: 「『家族のため』が、妻を追い詰めていた。——共働きのお金と時間の話」**
- フック: 著者の実体験(「家族のためのつもりが、妻に負担をかけていた」)を核にした**告白型の刃**。副業を足す前に「**夫婦の可処分時間はもうゼロ**」という直視。
- 埋める空白: b(共働きの時間の奪い合い構造・競合薄=狙い目)
- 分量: 約2.5万字 / 章: 時間の奪い合い→可処分時間の可視化→世帯年収を上げる順番→家事外注は投資→夫婦のキャリア設計→おわりに

#### 第5巻(逆説・燃え尽き)
- **タイトル: 「稼いでも稼いでも、なぜ楽にならないのか。——"もっと"の罠から抜ける話」**
- フック: **読者の痛みをそのまま問いにする**(「稼いでも楽にならない」)。"手放す技術"のような"欲求されない解決策"は避け、「この消耗の正体を知りたい/抜け出したい」という実際の欲求に当てる。
- 角度の再設定: 「足るを知る/手放す」でなく「**"もっと稼げば楽になる"という罠から抜ける**」。FIRE卒業・脱タイパの時流に接続。著者素材(完璧主義・未完プロジェクトへの執着・健康を崩しかけ)を核に。
- 埋める空白: d(稼げた側の燃え尽き)+ 追い風(FIRE卒業/メンパ)
- 分量: 約2万字 / 章: 途中で折れた話【一次情報】→"もっと"の終わらなさ→FIRE卒業と同じ構造→タイパの次はメンパ→足るを知る実践→おわりに

#### 第6巻(尖った刃・ROI/入口拡散)
- **タイトル: 「その副業、時給にすると300円かもしれない。——始める前の損得計算」**
- フック: **具体的な数字で刺す**(「300円」は例示・"かもしれない"で断定回避)。読むと自分の副業の時給を計算させられる。
- ⚠️ 執筆注意: 「300円」はフックの例示。本文では読者に**自分で計算させる**構成にし、特定の数字を事実として断定しない(`docs/07`)。
- 埋める空白: e(家族へのROIフレーム)/ 入口を安く広げるリードマグネット的役割
- 分量: 約2万字・低価格 or KU / 章: 時給の出し方→隠れコスト(税・時間・気力・家族)→割に合う/合わない境界→"やらない"という選択肢→その時間の別の使い道→おわりに

## 4. 尖らせ方の原則(炎上せず、刺す)

「尖った内容で掴む」を、規約・炎上リスクと両立させる:

1. **攻撃対象は"人"でなく"構造"**(「副業する奴はバカ」ではなく「割に合わない仕組み」を撃つ)。
2. **稼いだ当事者の資格で断言する**(評論でなく体験だから許される尖り)。
3. **主役は後悔、数字は控えめ**(自慢に転ぶと反感)。
4. **断定でなく告白の形**(「絶対やめろ」でなく「僕はやめた、理由はこうだ」)。
5. **必ず出口を示す**(「やめとけ」で終わらせず転職・昇給の道=無責任にしない)。
6. **誇大・違反ワードは使わない**(`docs/03`。「必ず」「誰でも」等はNG)。

## 5. 複製ルール(第2弾以降のスケール)

当たった型(おそらく第1巻 or 第6巻の"刃")を、次のマトリクスで横展開する。**当たってから複製する**(先に量産しない):

| 軸 | 例 |
|---|---|
| 対象 | 独身卒業組(40-50代)/ 新婚 / 未就学児親 / 中学受験期 |
| 職種 | エンジニア / 営業 / 公務員 / 医療職 |
| 年代 | 20代 / 30代 / 40代 |

例:「エンジニアの副業、時給いくらですか」「40代からの、静かな昇給」等。**型は同じ・中身は別物**(焼き直しは規約違反)。ここで初めて冊数が積み上がる。
