# 発売キット — 告知文・レビュー依頼・初速の段取り（全巻共通）

> 「原稿はできた、でも発売初日に人が来ない」を防ぐための"弾"。各巻の発売時にコピペで使う。
> 方針: 煽らない・誠実（`docs/06`/`08`）。SNSは著者の口調（カジュアル・自虐・直球＝`docs/09`）。規約遵守（レビューの対価・交換は禁止＝`docs/03`）。
> ［　］は各巻で差し替え（タイトル・URL）。第1巻＝『副業、やめました。』/第2巻＝『副業より、転職でした。』…。

---

## 0. 初速の段取り（KDP・発売の3日間）

1. **公開72時間はインデックス／カテゴリ反映待ち**。焦らない。まずKDP Selectの**無料キャンペーン（最大5日）**を発売直後に設定 → DL数を作る → 有料に戻す（`お作法`§5）。
2. **既存の読者から流す**（いちばん効く初速）: 第1巻の巻末・note・**LINE**から新刊へ。LINE配信は`第1巻_LINE配信文`の「新刊お知らせ」を使う。
3. **発売初日にSNS告知**（下記§1）。無料キャンペーン中に集中させる。
4. **3日目・7日目**に「読んでくれた人の声」を引用してもう一押し（自作自演はしない。実際の反応だけ）。
5. **30日/90日で`tracker`更新** → 伸びた巻に注力（`docs/02`）。

## 1. SNS告知文（X／発売初日）

### A. 発売ポスト（短・自虐フック）
```
新刊出しました。

『［タイトル］』

副業で1000万稼いで、700万溶かした僕が、
ドヤ顔のノウハウじゃなく、"遠回りの後悔"を全部書いた本です。

いま無料（Kindle Unlimited）でも読めます。
［Kindleリンク］
```

### B. 中身をチラ見せ（スレッド2投目）
```
この本、こんな話です👇

・［その巻の刺さる一文／数字。例:副業で月1万円すら稼げない人が48.1%］
・きれいな成功談はゼロ。冷や汗と後悔だけ
・でも、そこから抜けた"具体"まで書いた

「稼げ！」より「消耗するな」。そういう本です。
```

### C. note連動（無料で試し読み→本へ）
```
本の「まえがき＋第1章」を、noteで無料公開しました。
まずタダで、合うか確かめてください。
［note公開版リンク］
```

> ⚠️禁止: 「必ず」「誰でも」「稼げる」等の誇大（`docs/03`）。ランキング操作を狙う相互支援の呼びかけ。

## 2. 巻末・LINEでのレビュー依頼（正直な依頼のみ）

### 本文おわりに（既に各巻に収録済み・共通トーン）
```
もしこの本が、少しでもあなたの肩の荷を軽くできたなら、
ほんの一言でも構いません。感想（レビュー）を聞かせてもらえると、
本当に励みになります。あなたの一言が、次の一冊を書く力になります。
```

### LINEでの依頼（読了者向け・数日後に1回だけ）
```
先日は本を読んでいただき、ありがとうございました。
もし何か一つでも「刺さった」ところがあれば、
Amazonのレビューで、正直な感想を教えてもらえると嬉しいです。
（★の数より、あなたの一文が、これから読む人の助けになります）
▶ ［レビューページのリンク］
```

> ⚠️規約（`docs/03`）: レビューの**対価・割引・プレゼント・相互レビューは禁止**。「正直な感想を」だけ。低評価も歓迎する姿勢を崩さない。

## 3. 各巻の"刺さる一文／数字"ストック（告知Bで使う）

| 巻 | フック候補 |
|---|---|
| 第1巻 | 「稼げなかったから、じゃない。稼げた。そのうえで、割に合わなかった」 |
| 第2巻 | 「副業に注いだ数百時間はマイナス100万。正しく使った十数時間が、毎年300万を生んだ」 |
| 第3巻 | 「15年、昇給の負け組だった僕が、静かに上がっていく人を観察して盗んだ話」 |

| 第5巻 | 「その副業、時給にすると300円かもしれない。始める前に、電卓を」 |
| 第6巻 | 「稼いでも稼いでも楽にならないのは、"もっと"に終わりがないから」 |

## 4. クロス告知（シリーズを回す）
- 各巻の発売時、**前の巻の読者（LINE）**へ「続き出ました」。
- note各記事の末尾に、**関連巻**へのリンク（第2⇄第3は相互、第6→第1→第2/3の順路）。
- 無料note『借金100万円は、可愛い後輩の誘いから』は**シリーズ全体の入口**として常時回す。

> これがあると「発売＝既存読者に一斉に届く」状態になる。ゼロから毎回集め直さない（`docs/08`ファネル）。
