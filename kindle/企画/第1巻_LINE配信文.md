# 第1巻 LINE公式アカウント 配信文セット

> 巻末CTAで友だち追加した読者に送る文面。方針=売り込まない・信頼と地続きのものだけ（`docs/08`/`docs/09`）。絵文字は控えめ、著者の声のまま。
> ［　］は登録後に設定する箇所。

---

## 1. あいさつメッセージ（友だち追加直後・自動返信）

```
追加ありがとうございます。
本を読んでくださったんですよね。最後まで付き合ってくれて、本当にうれしいです。

さっそくですが、お礼に、本編に書ききれなかった特典を用意しました。

▼ 消耗せずに所得を上げる「最初の1歩」チェックリスト（PDF）
　［特典PDFのリンク］

僕が、副業で消耗した状態から、どうやって本業に賭け直して、年収を400万から700万に上げたのか。その"最初の一歩"だけを、明日から動ける形にまとめたものです。

連絡するのは、多くても月に数回です。
たまに、本に書ききれなかった「消耗せずにお金を増やす」話や、次の本のお知らせを、静かに送るくらいです。合わなければ、いつでもブロックしてください。それで大丈夫です。

まずは、上のPDFだけでも受け取ってくださいね。
```

## 2. 初回ブロードキャスト（追加から3〜4日後）

```
先日は、追加ありがとうございました。特典のチェックリスト、目を通してもらえたでしょうか。

一つだけ、あの本でいちばん伝えたかったことを、改めて。

「お金を増やすこと」より、「消耗しないこと」を先に置く。
たったこれだけで、選ぶ手段の順番が変わります。副業をもう一個足す前に、まず"いまの本業の価値を上げられないか"を考える。遠回りに見えて、これがいちばん速かったんです。

次の本『副業より、転職でした。』では、そのやり方——僕が業務委託から実績を作って、スカウトされて、自分から金額を提示して年収を上げるまでを、包み隠さず書いています。
出たら、ここでいちばん最初にお知らせしますね。

今日も、お疲れさまでした。
```

## 3. 第2巻リリース時のお知らせ（出たとき）

```
お待たせしました。
次の本『副業より、転職でした。』が出ました。

「副業の次に、具体的に何をすればいいのか」。
15年いた業界から、どうやって年収を400万から700万に上げたのか、その全ステップを書いています。第1巻を読んでくれたあなたにこそ、届けたい一冊です。

▼［第2巻のリンク］

（もし第1巻がよかったら、感想やレビューをもらえると、次を書く力になります。いつも、ありがとうございます）
```

## 運用メモ
- 配信は"低頻度"で。売り込みを混ぜない（`docs/08`=信頼が資産）。
- リッチメニュー（任意）: 「① 特典を受け取る」「② 第1巻を読む」「③ noteを読む」の3つでOK。
- 無料枠（月200通）で開始可能。反応を見て有料枠を検討。
- ［　］の差し替え: 特典PDFリンク / 第1巻・第2巻のKindleリンク / noteリンク。
