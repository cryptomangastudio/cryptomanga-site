# 意思決定ログ（ADR）

## D-001 「Xtoon型の課金サイトを作る」を棄却
- **決定**：自社課金プラットフォーム（サイト）を事業本体にするのはやめる。
- **理由**：キャラIP/グッズモデルでは、収益＝ライセンス/グッズ、集客＝SNS。課金サイトは収益・集客どちらのエンジンにもならず、決済リスク・開発コスト・二面市場の立ち上げ苦労だけを背負う。
- **サイトの残る役割**：①キャラのブランド入口（1枚もの）②ヒット後のECショップ。どちらも「後回し」。

## D-002 「AIタレント(疑似人格)」より「キャラクターIP(ちいかわ型)」を採用
- **決定**：Xtoon 7海型の疑似人格ではなく、オリジナルのマスコット的キャラIPを主軸にする。
- **理由**：①市場が桁違い（2.85兆円）②絵の綺麗さでなく感情設計＝自社の本業に依存③R18/決済/反AIの地雷を回避。
- **代償**：当たるかゼロかの分散が大きい → ポートフォリオ運営＋激安検証で対処。

## D-003 AIは「開示するが売りにしない」
- **決定**：AI利用は開示。ただしマーケの前面はキャラの魅力と人間の作家性。「AIで作った」を目玉にしない。
- **理由**：海外データで公表AIは高収益、炎上は"隠蔽疑惑"が引き金。隠してバレるのが最悪手。
- **棄却案**：（前回の暫定案）「AIを完全に裏方・非公表」→ バレると欺瞞認定で即死のため棄却。

## D-004 全年齢SFWで振り切る
- **決定**：R18はやらない。
- **理由**：Visa/Master決済停止の実例多数（DLsite/Fantia/FANZA）。全年齢なら1.6〜1.8兆円のライセンス経済・ブランドコラボ・世界市場・App Store が開く。

## D-005 ベンチマークの結論＝「かわいい×内向きの毒」に収束
- **決定**：5分野(日本かわいい系/海外AIネイティブ/毒・政治/収益フォーマット/アジア・西洋収集モデル)のベンチマークを洗い出し、勝ちの型を確定。詳細 `03-benchmarks.md` / `04-winnable-plays.md`。
- **勝ちの型**：かわいい×内向きの毒 → AIで多産→SNSで多打席→当たりだけ製品化 / 入口ほぼ0円(LINEスタンプ+POD+SNS4コマ)・増幅はライセンスアウト / AI開示・全年齢・完全オリジナル。
- **ピック(勝算順)**：①社会あるある擬人化(=政治の勝てる翻訳)★★★★★ ②感情ラベル×収集モデル ③開示AIの萌え設定化 ④実用スタンプで接触面。

## D-006 「政治×キャラIP」は"翻訳"して採用
- **決定**：政治を核にはしない。「政治」を「世代・労働・生活のあるある風刺(敵＝抽象的な社会)」に翻訳した Pick 1 を旗艦候補に。
- **理由**：政治を核にすると客層二分・ブランド離れ・IP制御喪失(Pepe型)・決済/検閲リスクで商業的に死ぬ(こくみんうさぎ/いしんのしし/Pepe/せんとくんで実証)。内向きの毒に変換すれば ★★★★★ で勝てる(おぱんちゅ/ちいかわ/でぶどり)。
- **鉄則**：毒は「読者に向けた鏡」ならOK、「誰かに向けた矢」ならNG。換金の見せ方(ワニ型炎上)を設計段階で織り込む。
