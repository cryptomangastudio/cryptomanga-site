# 事業の核（Thesis）

## 一文
**「AIネイティブのスパイラルキュート」** ——
日本のかわいい系キャラIPを、AIで安く速く量産検証し、当たりを2.85兆円のグッズ/ライセンス経済に載せる。
公表AIで、売りはAIでなく**キャラの魂**。全年齢SFWで世界も射程。

## なぜ勝てるか（構造的優位）
1. **希少資源を握る。** 画像がタダになった世界で希少なのは物語・世界観・共感設計＝CryptoMangaの本業。単純なマスコットほどAIの一貫性も出しやすい（二重に有利）。
2. **空白の車線。** 米中のAI勢は「アニメ動画/配信」「量産モーションコミック」に集中。**日本のかわいい系キャラ×グッズ/ライセンス**をAIネイティブでやる者がいない。
3. **地雷を回避。** 全年齢SFW → 決済リスク（Visa/Master停止）なし、1.6〜1.8兆円のライセンス経済とコラボ可。
4. **海外に実証テンプレがある。** Invisible Universe（$20M）、Toonstar（WME契約）＝「AIネイティブのキャラIPスタジオ」は成立する事業だと証明済み。

## 唯一にして最大の弱点
**「注目/発見」＝バズは確率的で保証できない。** neuro-samaもimmaも外れ値。中央値のキャラは埋もれる。
→ だから **ポートフォリオ運営（多く起こし、速く畳み、当たりに張る）** と **激安検証** で分散に対処する。

## やらないこと（Non-goals）
- Xtoon型の課金プラットフォーム（サイト）を作ること ← 収益/集客エンジンにならない
- R18 ← 決済爆弾・ブランド毀損
- 既存作家の画風の模倣 ← 炎上＋法的リスク
- 「AIで作りました」を"売り"にすること ← 新規性は切れる

## 勝ち筋の規律（3点だけ）
1. **開示≠死、欺瞞＝死。** AI利用は淡々と開示。隠してバレるのが最悪手。
2. **オリジナル厳守。** 完全新規デザイン。画風を寄せない。
3. **売るのはAIでなくキャラ。** 前に出すのは人間の作家性とキャラの魅力。
