# 法務チェックリスト（※専門家レビュー前提のたたき台）

> これは法的助言ではない。実際の商用展開前に弁護士レビュー必須。

## キャラデザイン
- [ ] 完全オリジナルか？（既存キャラ・実在人物・特定絵師の画風に酷似していないか）
- [ ] 名前・ロゴが既存商標と衝突しないか（J-PlatPatで確認）
- [ ] 商標出願を検討したか（キャラ名・ロゴ／ヒットの気配が出たら早めに）

## AI生成物まわり
- [ ] 使用ツールの利用規約で**商用利用OK**か（学習データ・出力の権利）
- [ ] 生成物が特定の既存作品を再現していないか（プロンプトに固有名を入れない）
- [ ] AI利用の開示方針を守っているか（`ai-disclosure-and-backlash.md`）

## グッズ/物販
- [ ] 景品表示法（おまけ・キャンペーンの景品上限：10万円 or 取引価額の20倍のいずれか低い方）
- [ ] 製造委託先との権利帰属（キャラの著作権はスタジオ保持）
- [ ] 特商法表示（EC開始時）

## ガチャ/ランダム施策（やるなら）
- [ ] 有償ランダム＋換金性は**賭博罪リスク**。運営は二次流通の買取を保証しない設計にする
- [ ] コンプガチャ的な射幸設計にしない
- [ ] 景表法の景品上限を守る

## ライセンスアウト
- [ ] 契約書に権利範囲・地域・期間・ロイヤリティ率・品質管理（監修権）を明記
