# 90日 検証プレイブック（1体あたり）

「本当に当たるか」を机上でなく、安く・反証可能に確かめるための実験手順。
※ decision-guardrails スキルの精神：**先に判断基準と失敗ラインを引いてから実行**する。

## 前提
- サイトは不要。要るのは **X / TikTok アカウント** と **LINEスタンプ**。
- 広告費ゼロでオーガニックの反応を見る（バズの再現性＝我々の唯一の弱点をここで測る）。

## 準備（Day 0）
- [ ] `characters/<name>/character-bible.md` 完成（人間が書く）
- [ ] キャラ設計＆キャラシート作成（`production/pipeline.md`）
- [ ] 4コマを **10〜15本ストック**（切らさず投下できる分）
- [ ] LINEスタンプ 1セット（8〜40個）を審査提出
- [ ] AI開示の一文をプロフィールに記載（`ai-disclosure-and-backlash.md`）

## 実行（Day 1–90）
- [ ] X/TikTokに **週3〜5本**投下（同じ時間帯で固定）
- [ ] 毎週 `validation/tracker.md` に数値を記録

## 測る指標（バズは「保存/シェア」が先行指標。フォロワーは遅行指標）
| 指標 | なぜ見るか |
|---|---|
| 保存率・シェア率 | キャラが"刺さって"いるかの最速シグナル |
| フォロワー純増（オーガニック） | 継続的な支持 |
| LINEスタンプ販売数 | 最初の"財布を開く"検証（最安のグッズ検証） |
| コメントの質 | キャラへの愛着/二次創作の芽 |

## 判定（Day 90・先に決めておく）
### ✅ 成功ライン（→ 拡張フェーズへ：書籍/グッズ/ライセンス）
- オーガニックでフォロワーが右肩上がり（例：月次で純増が加速）
- 保存/シェアが特定の投稿で跳ねる（＝再現できるツボがある）
- LINEスタンプが一定数売れる（＝財布が開く）

### ✖ 失敗ライン（→ そのキャラは畳む。学びを次の設計へ）
- 全力の設計で保存/シェアが伸びない
- どの4コマも横並びで反応がない（＝共感フックが弱い）

### 判断の心得
- 1体の失敗＝事業の失敗ではない。**ポートフォリオで打率を上げる**前提。
- ただし「全力の1体すら牽引できない」なら、モデルか実行力の見直し。
