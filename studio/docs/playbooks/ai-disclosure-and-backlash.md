# AI開示 & 反AI炎上回避プレイブック

## 大原則
> **観客は「開示」を許す。許さないのは「欺瞞」。隠してバレるのが即死。**

## やること（DO）
- [ ] プロフィール/概要欄に**AI利用を淡々と一文**で明記（隠さない、が言い訳もしない）。
  - 例：「制作にAI作画を用いています。物語・キャラクター設計は[スタジオ名]の人間が担当。」
- [ ] **人間の作家性を本物にし、前に出す**。物語・世界観・共感フックは人間が作る（伊藤園型）。
- [ ] **完全オリジナルデザイン**。既存キャラ・実在人物・特定絵師の画風に寄せない。
- [ ] 韓国など**AI表示義務**の市場に出す時はラベル要件を確認（規制は「開示が標準」へ動いている）。

## やらないこと（DON'T）
- ✗ 「人間が手描きした」と偽る（バレたら欺瞞認定＝プリキュア/Naver型の即死）。
- ✗ 「AIで作りました！」を"売り"の中心にする（新規性は切れる／反AIを自ら誘発）。
- ✗ 生きている絵師の画風・特定作品の要素を模倣（炎上＋法的リスク）。
- ✗ 実在タレントに酷似した顔（集英社さつきあい型のパブリシティ権問題）。

## 炎上の引き金（過去事例からの逆算）
| 事例 | 引き金 | 学び |
|---|---|---|
| プリキュア公式 | AI利用の**疑い**だけ | 疑われた時点で燃える → 先に開示して疑惑の芽を消す |
| Naver「Knight King」 | AI**隠し**疑惑でボイコット（評価1.94） | 隠蔽＝最悪。プラットフォーム巻き込みで大炎上 |
| 集英社「さつきあい」 | 実在酷似＋生成過程の不透明 | オリジナル＆透明性 |

## 炎上時の初動（起きたら）
1. 事実を短く開示（何にAIを使い、何を人間がやったか）。
2. 生きている絵師の模倣が事実なら即取り下げ＋謝罪。事実でないなら証拠（人間の制作過程）を示す。
3. 感情的に反論しない。「規律3点（開示/オリジナル/キャラ優先）」を守れていれば守勢に立てる。
